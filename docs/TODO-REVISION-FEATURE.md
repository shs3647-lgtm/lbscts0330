## 📧 메일 발송 규칙 (REV-003, REV-004)

```
발송 시점:
- D+0: 일자 초과 당일 (첫 번째 발송)
- D+3: 3일 후 (두 번째 발송)
- D+5: 5일 후 (세 번째 발송)
- D+10: 10일 후 (네 번째/최종 발송)

조건:
- 상태가 '진행' 또는 '지연'인 경우만 발송
- 상태가 '승인' 또는 '완료'이면 발송 중지
- 동일 건에 대해 최대 4회까지만 발송
```
