## 3. 이론적 계산: PFD 렌더링

### 3.2 PFD 컬럼 매핑 (CP → PFD)

| CP 컬럼 | PFD 컬럼 | 데이터 수 |
|--------|---------|----------|

## 5. ⚠️ 문제 발견: 지그위치

### 문제
- R34: 공정=30(용접), **제품특성=지그위치**
- 하지만 "지그위치"는 **공정특성**이어야 함 (JIG 관련)

### 원인
- Excel 원본에서 지그위치가 Col10(제품특성)에 입력됨
- 파싱 로직이 원본 그대로 저장

### 해결책
1. **Excel 원본 수정**: 지그위치를 Col11(공정특성)으로 이동
2. **또는 특성 분류 로직 추가**: "지그"가 포함된 값은 공정특성으로 자동 분류

## 9. 테스트 계획

### Phase 1: 데이터 정확성 검증
1. Excel 파싱 후 정확한 행 수 확인 (108개)
2. 제품특성 값 있는 행 수 확인 (21개)
3. 공정특성 값 있는 행 수 확인 (9개)

### Phase 2: 위치 정확성 검증
1. 지그위치가 제품특성에 있는지 확인 → 공정특성으로 이동 필요
2. RPM이 제품특성과 공정특성 둘 다 있는지 확인

### Phase 3: 연계성 검증
1. CP 행 수 = PFD 행 수
2. CP 제품특성 = PFD 제품특성
3. CP 공정특성 = PFD 공정특성
4. FMEA L2 = CP 고유 공정 수

### Phase 4: 렌더링 검증
1. 공정번호 병합 그룹 수: 8개
2. 제품특성/공정특성은 병합하지 않음
