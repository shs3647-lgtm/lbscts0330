### API 명세

| API | Method | 경로 | 설명 |
|-----|--------|------|------|
| 생성 | POST | `/api/fmea` | 새 프로젝트 생성 |
| 조회 | GET | `/api/fmea?id={id}` | 프로젝트 상세 조회 |
| 목록 | GET | `/api/fmea/list` | 프로젝트 목록 |
| 삭제 | DELETE | `/api/fmea?id={id}` | 프로젝트 삭제 |

### DB 스키마

```prisma
model FMEAProject {
  id            String   @id
  projectName   String
  productName   String?
  modelYear     String?
  customerName  String?
  supplierName  String?
  fmeaType      String   // PFMEA, DFMEA
  status        String   // draft, active, completed
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
}
```
