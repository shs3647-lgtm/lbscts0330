## 1️⃣ PFD ↔ PFMEA 연계

### 연계 포인트

| 항목 | PFD | PFMEA | 동기화 방향 | API |
|------|-----|-------|------------|-----|
| 공정번호 | `processNo` | `l2.no` | PFD → PFMEA | `/api/sync/structure` |
| 공정명 | `processName` | `l2.name` | PFD → PFMEA | `/api/sync/structure` |
| 공정순서 | `order` | `l2.order` | PFD → PFMEA | `/api/sync/structure` |

### DB 스키마 매핑

```
PFD.ProcessFlowItem          PFMEA.FMEAWorksheet
├── processNo       →        ├── l2Structures[].no
├── processName     →        ├── l2Structures[].name
└── order           →        └── l2Structures[].order
```

### 테스트 체크리스트

| TC ID | 테스트 케이스 | 전제조건 | 테스트 단계 | 예상 결과 | 상태 |
|-------|--------------|----------|------------|----------|------|
| TC-PFD-PFMEA-001 | PFD에서 PFMEA 공정 Import | PFD에 공정 3개 등록 | 1. PFMEA 워크시트 열기<br>2. 공정선택 모달<br>3. PFD Import 클릭 | PFMEA L2에 3개 공정 추가 | ⬜ |
| TC-PFD-PFMEA-002 | PFD 공정 변경 시 PFMEA 동기화 | PFD-PFMEA 연결 상태 | 1. PFD 공정명 변경<br>2. PFMEA 열기 | PFMEA L2 공정명 변경됨 | ⬜ |
| TC-PFD-PFMEA-003 | PFD 공정 삭제 시 PFMEA 영향 | PFD-PFMEA 연결 상태 | 1. PFD 공정 삭제<br>2. PFMEA 열기 | 경고 또는 참조 유지 | ⬜ |
| TC-PFD-PFMEA-004 | PFMEA 공정 수동 추가 후 PFD 영향 | PFD-PFMEA 연결 상태 | 1. PFMEA에서 공정 추가<br>2. PFD 열기 | PFD 영향 없음 (단방향) | ⬜ |

## 2️⃣ PFMEA → CP 연계

### 연계 포인트

| 항목 | PFMEA | CP | 동기화 방향 | API |
|------|-------|-----|------------|-----|
| 공정번호 | `l2.no` | `processNo` | PFMEA → CP | `/api/sync/cp` |
| 공정명 | `l2.name` | `processName` | PFMEA → CP | `/api/sync/cp` |
| 제품특성 | `l2.productChars` | `productChar` | PFMEA → CP | `/api/sync/cp` |
| 공정특성 | `l3.processChars` | `processChar` | PFMEA → CP | `/api/sync/cp` |
| 특별특성 | `CC/SC 플래그` | `specialChar` | PFMEA → CP | `/api/sync/cp` |
| 검출방법 | `currentControl.detection` | `detectionMethod` | PFMEA → CP | `/api/sync/cp` |
| 심각도 | `riskData.S` | `severity` | PFMEA → CP | `/api/sync/cp` |

### DB 스키마 매핑

```
PFMEA.FMEAWorksheet              CP.ControlPlan
├── l2Structures[]               ├── items[]
│   ├── no              →        │   ├── processNo
│   ├── name            →        │   ├── processName
│   └── productChars[]  →        │   ├── productChar
├── l3Structures[]               │   
│   └── processChars[]  →        │   ├── processChar
├── riskData                     │
│   └── S               →        │   └── severity
└── currentControls[]            │
    └── detection       →        └── detectionMethod
```

### 테스트 체크리스트

| TC ID | 테스트 케이스 | 전제조건 | 테스트 단계 | 예상 결과 | 상태 |
|-------|--------------|----------|------------|----------|------|
| TC-PFMEA-CP-001 | PFMEA 특성 → CP 전달 | PFMEA에 특별특성 등록 | 1. CP 열기<br>2. PFMEA Import | 특별특성 CP에 표시 | ⬜ |
| TC-PFMEA-CP-002 | PFMEA 검출방법 → CP 전달 | PFMEA에 검출방법 입력 | 1. CP 열기<br>2. 동기화 | 검출방법 CP에 표시 | ⬜ |
| TC-PFMEA-CP-003 | PFMEA S=9,10 → CP 자동 표시 | PFMEA에서 S=9 설정 | 1. CP 열기 | 해당 항목 강조 표시 | ⬜ |
| TC-PFMEA-CP-004 | PFMEA 최적화 조치 → CP 반영 | PFMEA 최적화 완료 | 1. CP 열기<br>2. 관리방법 확인 | 새 관리방법 표시 | ⬜ |
