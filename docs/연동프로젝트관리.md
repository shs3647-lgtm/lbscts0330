# 연동 프로젝트 관리 (중앙집중식)

## 📋 개요

APQP → PFMEA → PFD → CP 연동 관계를 **한 곳(ProjectLinkage 테이블)**에서 관리합니다.

### 기존 문제점
- 각 앱(PFMEA, PFD, CP)에서 개별적으로 연동 정보 저장
- 불일치, 누락, 꼬임 문제 발생
- 한쪽에서 수정하면 다른 쪽에 반영 안 됨

### 해결책
- **중앙집중식 `ProjectLinkage` 테이블**
- 모든 연동 관계를 한 곳에서 관리
- 각 앱은 이 테이블에서 **조회만** 하면 됨

---

## 📊 데이터베이스 스키마

```prisma
model ProjectLinkage {
  id          String   @id @default(uuid())
  apqpNo      String?  // 상위 APQP 번호
  pfmeaId     String?  // PFMEA ID
  pfdNo       String?  // PFD 번호
  cpNo        String?  // CP 번호
  projectName String?  // 프로젝트명
  customerName String? // 고객명
  linkType    String   @default("auto")    // 'auto' | 'manual'
  status      String   @default("active")  // 'active' | 'deleted'
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
  createdBy   String?
}
```

### 1:N 관계 지원

하나의 PFMEA에서 여러 CP가 생성될 수 있습니다. 이 경우 **여러 레코드**로 표현합니다:

| apqpNo | pfmeaId | pfdNo | cpNo |
|--------|---------|-------|------|
| apq26-001 | pfm26-m001 | pfdl26-p001 | cpl26-p001 |
| apq26-001 | pfm26-m001 | pfdl26-p002 | cpl26-p002 |
| apq26-001 | pfm26-m001 | pfdl26-p003 | cpl26-p003 |

## 🔗 API 엔드포인트

### 1. 연동 정보 조회
```
GET /api/project-linkage
GET /api/project-linkage?pfmeaId=pfm26-m001
GET /api/project-linkage?pfdNo=pfdl26-p001
GET /api/project-linkage?cpNo=cpl26-p001
```

### 2. 연동 정보 저장/수정
```
POST /api/project-linkage
Body: {
  apqpNo: "apq26-001",
  pfmeaId: "pfm26-m001",
  pfdNo: "pfdl26-p001",
  cpNo: "cpl26-p001",
  projectName: "테스트 프로젝트",
  customerName: "현대자동차"
}
```

### 3. 기존 데이터 동기화
```
POST /api/project-linkage/sync
```
- PFMEA, PFD, CP에 저장된 기존 연동 정보를 ProjectLinkage 테이블로 마이그레이션

### 4. 동기화 상태 확인
```
GET /api/project-linkage/sync
```

## 🔄 워크플로우

### 프로젝트 생성 시
1. PFMEA 생성 → ProjectLinkage에 레코드 생성 (pfmeaId만)
2. PFD 연동 시 → 해당 레코드에 pfdNo 추가
3. CP 생성 시 → 해당 레코드에 cpNo 추가 (또는 새 레코드 생성 for 1:N)

### 리스트 화면에서 조회 시
- PFMEA 리스트: `GET /api/project-linkage?pfmeaId=xxx` → 연동 PFD, CP 표시
- PFD 리스트: `GET /api/project-linkage?pfdNo=xxx` → 연동 PFMEA, CP 표시
- CP 리스트: `GET /api/project-linkage?cpNo=xxx` → 연동 PFMEA, PFD 표시
