# 도움말(Help) 버튼 배치 지침서

> **작성일**: 2026-03-13  
> **적용 범위**: FMEA On-Premise 전체 화면  
> **목적**: 도움말 버튼 중복 방지 및 일관된 UX 제공

---

## 1. 원칙

### 1.1 단일 진입점 원칙
- **상단 액션바(Create/Edit/Save 영역)에는 도움말 버튼을 배치하지 않는다**
- 도움말은 해당 **섹션 헤더 우측**에 1개만 배치한다
- 같은 화면에서 동일 도움말을 여는 버튼이 2개 이상 있으면 안 된다

### 1.2 문맥형 도움말 (Contextual Help)
- 각 섹션의 도움말은 해당 섹션의 내용만 설명해야 한다
- 일반적인 "개요(overview)" 도움말은 **TopNav의 Help/매뉴얼**에서 담당한다

### 1.3 계층 구분

| 레벨 | 위치 | 도움말 유형 | 예시 |
|------|------|------------|------|
| **Global** | TopNav (최상단 네비게이션) | Help 채팅, 매뉴얼 검색 | 💬 Help, 📖 매뉴얼(Manual) |
| **Section** | 섹션 헤더 우측 | 해당 섹션 필드/기능 설명 | 기획 및 준비 → 도움말(Help) |
| **Inline** | 필드 옆 `?` 아이콘 | 개별 필드 도움말 (툴팁) | HelpIcon compact 모드 |

## 2. 배치 규칙

### 2.1 허용 위치 (O)

```
┌─────────────────────────────────────────────┐
│ TopNav: 💬 Help │ 📖 매뉴얼     [Global]   │
├─────────────────────────────────────────────┤
│ 액션바: ➕새로 작성 │ ✏️편집 │ 💾저장       │ ← 도움말 배치 ❌
├─────────────────────────────────────────────┤
│ 섹션 헤더: 기획 및 준비        [도움말] ✅  │ ← 섹션 우측
│ ┌──────────────────────────────────────┐    │
│ │ 필드들 ...                           │    │
│ └──────────────────────────────────────┘    │
│ 섹션 헤더: FMEA 기초정보등록   [도움말] ✅  │ ← 섹션 우측
│ ┌──────────────────────────────────────┐    │
│ │ Master/Family/Part 탭 ...            │    │
│ └──────────────────────────────────────┘    │
│ 섹션 헤더: 기초정보 템플릿     [도움말] ✅  │ ← 섹션 우측
│ ┌──────────────────────────────────────┐    │
│ │ 미리보기 테이블 ...                  │    │
│ └──────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

### 2.2 금지 위치 (X)

| 위치 | 사유 |
|------|------|
| 상단 액션바 (Create/Edit/Save 옆) | 핵심 액션 버튼과 혼재 → 클릭 실수 유발, 중복 |
| 테이블 내부 셀 | 데이터 영역 침범 |
| 모달 내부 (단, 복잡한 입력 모달은 예외) | 모달은 자체 설명이 충분해야 함 |

### 2.3 스타일 통일

```tsx
// 섹션 도움말 버튼 — 표준 스타일
<button
  onClick={() => openHelp('sectionKey')}
  className="px-1.5 py-0.5 bg-yellow-400 text-[#00587a] text-[9px] font-bold rounded hover:bg-yellow-300 transition-colors"
  title="섹션 설명 도움말"
>
  도움말(Help)
</button>

// 인라인 도움말 — HelpIcon compact 모드
<HelpIcon title="필드 설명" compact />
```

## 3. 화면별 도움말 배치 현황

### PFMEA 등록 화면 (`/pfmea/register`)

| 섹션 | 도움말 키 | 위치 | 내용 |
|------|----------|------|------|
| 기획 및 준비 | `fields` | 섹션 헤더 우측 | FMEA 유형, 담당자, 고객 등 필드 설명 |
| FMEA 기초 정보등록 | `mfp` | 섹션 헤더 우측 | M/F/P 구조, Import 프로세스 설명 |
| 기초정보 템플릿 | HelpIcon | 패널 헤더 우측 | 템플릿 생성/다운로드/편집 가이드 |
| CFT 구성 | `cft` | 섹션 헤더 우측 | CFT 역할/담당자 등록 안내 |
| ~~상단 액션바~~ | ~~overview~~ | ~~제거됨~~ | ~~TopNav Help로 대체~~ |

### 워크시트 화면 (`/pfmea/worksheet`)

| 섹션 | 위치 | 내용 |
|------|------|------|
| 워크시트 테이블 | 섹션 헤더 우측 | 열/셀 편집 가이드 |

### 기타 화면

| 화면 | 권장 배치 |
|------|----------|
