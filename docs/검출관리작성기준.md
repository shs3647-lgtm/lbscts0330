## 1. 검출관리(DC)의 정의와 목적

### 1.1 DC의 본질적 정의

> **검출관리(DC, Detection Control)**  
> 공정 또는 제품에 발생한 고장·불량을 후속 공정 또는 고객에게 유출되기 전에  
> **검사·측정·모니터링 장치로 직접 찾아내는 수단**

DC 개선대책은 반드시 **구체적인 검사·측정·모니터링 장치명**을 포함해야 한다.  
장치명이 없는 개선대책은 검출관리로 인정하지 않는다.

### 1.2 DC와 PC의 핵심 구분

| 구분 | 핵심 질문 | 작용 시점 | 장치 여부 |
|------|---------|---------|---------|
| **예방관리(PC)** | 고장이 생기지 않게 막는가? | 고장 발생 **이전** | 장치 불필요 |
| **검출관리(DC)** | 발생한 불량을 장치로 잡아내는가? | 고장 발생 **이후** | **장치명 필수** |

> **핵심 원칙**: 검사·측정 방법의 관리 수준 개선(주기 단축, 빈도 강화, 기준 재설정 등)도  
> **해당 검사·측정·모니터링 장치명을 명시**하면 DC 개선대책으로 인정된다.

## 2. DC 개선대책 작성 필수 원칙

### 2.1 4대 필수 요소

모든 DC 개선대책 1문장에 아래 4가지 요소가 포함되어야 한다.

```
① 검사·측정·모니터링 장치명   → 무엇으로 검출하는가  (필수, 생략 불가)
② 검사 대상 특성             → 무엇을 측정·판정하는가
③ 검사 범위/빈도             → 전수 / 샘플링 / Lot별 / Shift별
④ 불합격(NG) 처리 방법       → NG 시 어떻게 처리하는가
```

### 2.2 장치명 명시 기준

장치명은 **일반명칭이 아닌 구체적인 장치 종류**까지 기재한다.

| ❌ 불충분한 장치명 | ✅ 충분한 장치명 |
|----------------|---------------|
| "계측기" | "마이크로미터", "버니어캘리퍼스", "에어게이지" |
| "검사기" | "비전검사기", "기능검사기", "체결검사기", "리크검사기" |
| "측정기" | "도막두께계(마그네틱식)", "색차계", "소음측정기" |
| "시험기" | "항온항습 시험기(85℃/85%RH)", "염수분무 시험기(SST)", "UTM 인장시험기" |
| "게이지" | "나사산 GO/NO-GO 게이지", "링게이지", "플러그게이지" |
| "CMM" | "CMM(3차원 측정기)" — CMM은 약어이므로 풀네임 병기 |

### 2.3 관리방법 개선도 DC로 인정되는 경우

검사·측정 방법의 단순한 관리 수준 개선(주기 단축, 빈도 강화 등)은  
**해당 장치명을 명시**하면 DC 개선대책으로 인정된다.

```
[인정 예시]
"마이크로미터 Gage R&R 불합격 시 재측정 프로세스 즉시 가동"       ← 마이크로미터 명시 → DC ✅
"CMM(3차원 측정기) 프로브 교체 후 기준구 재교정 및 재측정 검증"    ← CMM 명시 → DC ✅
"리크 검사기 마스터 샘플 일일 감도 검증, 이탈 시 Shift 전수 재검"  ← 리크 검사기 명시 → DC ✅
"비전검사기 조명 광도 모니터링, 기준 이탈 시 라인 정지 인터락"      ← 비전검사기 명시 → DC ✅

[불인정 예시]
"Gage R&R 불합격 시 재측정 프로세스"   ← 장치명 없음 → DC ❌ (장치명 추가 필요)
"검사 빈도 강화"                       ← 장치명 없음 → DC ❌
"마스터 샘플 일일 검증"                ← 장치명 없음 → DC ❌
```

## 3. DC와 PC 구분 기준

### 3.1 판별 플로우차트

```
개선대책 내용 검토
        │
        ▼
┌─────────────────────────────────────────┐
│ Q1. 구체적인 검사·측정·모니터링            │
│     장치명이 명시되어 있는가?              │
└──────────────┬──────────────────────────┘
      YES      │        NO
        ▼                ▼
┌──────────────────┐   장치명 추가 후
│ Q2. 발생한 불량을 │   DC로 재작성
│ 검출하는 목적인가?│
└───────┬──────────┘
  YES   │    NO
    ▼         ▼
검출관리(DC)  예방관리(PC)
→ DC 필드     → PC 필드
```

### 3.2 DC vs PC 키워드 분류

| **DC 인정 ✅** (장치명 명시 시) | **PC 전용 ❌** (DC 필드 등록 불가) |
|------------------------------|--------------------------------|
| 마이크로미터·버니어캘리퍼스 전수 측정 | 검교정 주기 단축 (장치명 없는 경우) |
| 비전검사기 OK/NG 마스터 샘플 검증 | 작업자 교육 강화 |
| CMM(3차원 측정기) Gage R&R 실시 | 작업표준서 개정 |
| 리크 검사기 감도 업그레이드 | 4M 변경관리 |
| 도막두께계 전수 측정 | DOE 기반 조건 최적화 |
| 색차계 L*a*b* 정량 판정 | 설계 안전율 상향 (그 자체로는) |
| 소음측정기 + 방음 부스 전수 측정 | 도면 GD&T 표기 명확화 (그 자체로는) |
| 체결검사기 Poka-Yoke 인터락 | 예방보전 PM 주기 단축 |
| SPC 시스템(측정기 연동) 이상 자동감지 | 공정능력(Cpk) 조건 최적화 활동 |
| EOL 기능검사기 마스터 샘플 검증 | PPAP 절차 보완 (성적서 없는 경우) |
| 항온항습 시험기 성적서 Gate 관리 | 표면 처리 사양 변경 (그 자체로는) |

## 4. 검출도(D) 등급 기준

### 4.1 D값 등급표 (AIAG-VDA 보수적 기준)

| D | 등급명 | 검출관리 수준 | 대표 검사·측정·모니터링 장치 |
|---|--------|-------------|--------------------------|
| **1** | 거의 확실 | 자동 인터락/Poka-Yoke | 불량품 자동 차단 장치 (사람 개입 없음) |
| **2** | 매우 높음 | 기계 검출 + 사람 확인 | 자동검출 장치 + 작업자 확인 |
| **3** | 높음 | 자동측정/SPC+인터락 | 자동측정기 + 라인 정지 인터락, GO/NO-GO 전수 |
| **4** | 보통 (높음) | 전용 검사기·게이지 전수 | 도막두께계·색차계·CMM·리크검사기 등 전용 장비 전수 |
| **5** | 보통 | 검교정 보증 계측기 | 마이크로미터·버니어캘리퍼스(검교정 유효) |
| **6** | 낮음 | 전수 육안검사 | 한계견본 + 조명·확대경 기준 전수 육안 |
| **7** | 매우 낮음 | 일반 육안·비전(MSA 미입증) | 비전검사(MSA 미입증), 일반 육안 샘플 |
| **8** | 거의 불가 | 간접 확인 | 간접 모니터링, 샘플 육안 |
| **9** | 불가 | 검출관리 미흡 | 검출 방법 부재 |
| **10** | 검출 불가 | 검출관리 없음 | 검출 수단 전무 |

### 4.2 D값 특수 규칙

| 검출 방법 | D값 | 근거 |
|----------|-----|------|
| Poka-Yoke 자동 차단 장치 | **D=1** | 사람 개입 없는 100% 자동 차단 |
| 자동측정기 + 라인 정지 인터락 | **D=2~3** | 즉각 자동 대응 |
| GO/NO-GO 게이지 전수 + 자동 NG 배출 | **D=3** | 전용 게이지 전수 자동 판정 |
| 도막두께계·색차계·CMM 전수 | **D=4** | 전용 장비 전수 측정 |
| 비전검사기 + MSA 입증 | **D=3~4** | MSA 검증된 자동검사 |
| 비전검사기 단독 (MSA 미입증) | **D=7** | 보수적 평가 |
| 마이크로미터·버니어캘리퍼스 (검교정 유효) | **D=5** | 검교정 보증 수동 계측 |
| 한계견본 기준 전수 육안검사 | **D=6** | 사람 의존 최선 |
| **육안검사 모든 경우** | **D≥6** | D=1~5 절대 불가 |

| **도막·도금 두께** | 도막두께계(마그네틱·와전류식) | D=3~4 | 전수/AQL |
| **색상 편차** | 색차계 (L*a*b* 정량 판정, ΔE 기준) | D=4 | 전수/AQL |
| **용접 불량** (크랙·기공·강도) | 초음파탐상기(UT), X-ray 비파괴검사기, 용접 비전검사기 | D=3~4 | 전수/Lot |
| **중량·무게 불량** | 자동 중량선별기, 전자저울·발란스 측정기(검교정) | D=3~4 | 전수 |
| **나사산 불량** | 나사산 GO/NO-GO 게이지 | D=3 | 전수 |
| **리크·기밀 불량** | 리크검사기(cc/min 감도 기준), 수압시험기 | D=3~4 | 전수 |
| **소음·진동 이상** | 소음측정기(방음 부스 병용), NVH 분석기 | D=4 | 전수 |
| **전기·전자 기능 이상** | 기능검사기(전류·전압 자동 판정), ICT, FCT | D=3~4 | 전수 |
| **형상·복합 치수** | CMM(3차원 측정기), 3D 스캐너 | D=4 | Lot 샘플 |
| **SPC 이상 패턴** | SPC 자동 모니터링 시스템(측정기기 연동) + 인터락 | D=3 | 전수 연속 |

| **PDA·바코드 스캔** | PDA 바코드 스캐너 + 스캔 완료 인터락 | D=3 | 전수 |

## 6. 오류 유형별 수정 기준 및 변환 예시

> LLD 검출관리 데이터 점검 시 아래 5가지 오류 유형으로 분류 후 수정한다.

### 6.1 오류 유형별 수정 방향

| 오류 유형 | 판단 기준 | 수정 방법 |
|----------|---------|---------|
| **① 장치명 누락** | DC 내용은 맞으나 장치명이 없는 경우 | 5.1항 표에서 장치명 선택 후 문장 앞에 추가 |
| **② PC 혼입** | 장치명 없이 교육·조건최적화·설계변경만 기술 | 해당 내용 PC 필드로 이동, DC는 검사장치 기반 재작성 |
| **③ 검사 빈도만 기술** | "Lot당→시간당 검사" 처럼 빈도 관리만 기술 | 장치명 + 빈도 개선 + NG 처리 조합으로 재작성 |
| **④ 설비 관리만 기술** | 알림 시스템, 교체 주기만 기술 | 해당 설비(검사장치) 이상 감지 후 검증 방법 명시 |
| **⑤ D값 불일치** | 기술된 방법 대비 D값 과소·과대 | 4.1항 D값 등급표 기준으로 재산정 |

### 6.2 변환 예시 (전·후 비교)

#### ① 장치명 누락 → 추가

| 오류 (장치명 없음) | 올바른 수정 (장치명 명시) |
|----------------|----------------------|
| "Gage R&R 불합격 시 재측정 프로세스 즉시 가동" | "**마이크로미터** Gage R&R 불합격 시 **마이크로미터** 재측정 프로세스 즉시 가동" |
| "마스터 샘플 매 Shift 기준 검증" | "**EOL 기능검사기** 마스터 OK/NG 경계 샘플 매 Shift 기준 검증" |
| "감도 업그레이드 및 일일 검증" | "**리크 검사기** 감도 업그레이드(10cc→1cc/min) 및 **리크 검사기** 마스터 리크 샘플 일일 감도 검증" |

#### ② PC 혼입 → DC 교체

| 오류 (PC 성격) | 올바른 수정 (DC로 교체) |
|-------------|-------------------|
| "검교정 이력 전산화 및 만료 7일 전 자동알림 시스템 도입" | "**마이크로미터·버니어캘리퍼스** 검교정 만료 시 RED TAG 부착 → 검교정 완료 대체 계측기로 전수 재측정 실시" |
| "검사원 검사 기준 교육 강화 및 MSA 분기→월1회" | "**외관검사 한계견본(OK/NG 한계샘플)** 기준 신규 제정·공정 게시 → 한계견본 기준 전수 육안검사 실시" |
| "DOE 기반 양산 조건 최적화 및 Cpk 1.67 달성 후 이관" | "공정 **측정기기(마이크로미터·게이지류)** 데이터 자동 수집 **SPC 시스템** → Cpk<1.33 시 즉시 라인 정지 인터락" |

#### ③ 검사 빈도만 기술 → 장치명 + 처리 추가

| 오류 (빈도만 기술) | 올바른 수정 |
|----------------|----------|
| "나사산 게이지 검사 빈도 증가(Lot당→시간당)" | "**나사산 GO/NO-GO 게이지** 전수 검사(빈도 Lot당→시간당 강화) → NO-GO 통과 NG 시 자동 배출 인터락" |
| "Zero점 확인 빈도 Lot→매 1시간 강화" | "**발란스 측정기(전자저울)** Zero점 매 1시간 확인 → Zero점 허용기준(±0.1g) 초과 시 자동 경보 및 해당 시간대 전수 재측정" |

#### ④ 설비 관리만 기술 → 장치명 + 검증 방법 추가

| 오류 (설비 관리만) | 올바른 수정 |
|----------------|----------|
| "용접 전극 팁 드레싱 주기 자동 카운트 및 교체 알림" | "**용접부 인라인 비전검사기**(고해상도 카메라+링조명) 전수 외관검사 → 전극 마모로 발생한 언더컷·크랙 자동 NG 판정" |
| "EOL 검사기 센서 정기점검 주기 단축(분기→월)" | "**EOL 기능검사기** 마스터 OK/NG 경계 샘플 매 Shift 기준 검증 → 판정 불일치 시 해당 Shift 전수 재검사 실시" |
| "CMM 프로브 사용 500회 시 자동 교체 알림" | "**CMM(3차원 측정기)** 프로브 교체 후 기준구(Reference Sphere) 재교정 → 교체 전후 동일 기준 부품 CMM 재측정으로 편차 허용공차(±3σ) 이내 검증" |

## 7. 등록 전 체크리스트

```
□ 1. 검사·측정·모니터링 장치명이 구체적으로 명시되어 있는가?
      버니어캘리퍼스 / 마이크로미터 / 에어게이지 / CMM(3차원 측정기) /
      비전검사기 / AOI / 도막두께계(마그네틱식) / 색차계 / 소음측정기 /
      체결검사기 / 리크검사기 / 기능검사기 / 경도계 / UTM 인장시험기 /
      항온항습 시험기 / SST 시험기 / 내구시험기 / PDA 바코드 스캐너 등
      → 없으면: 5.1항 표에서 고장형태에 맞는 장치명 선택 후 반드시 추가

□ 2. 검사 대상 특성이 명시되어 있는가?
      도금 두께 / 체결 토크값 / L*a*b* 색상값 / 절단 길이 / 경도(HRC) 등
      → 없으면: 고장형태(FM)와 직결되는 측정 특성 추가

□ 3. 검사 범위/빈도가 명시되어 있는가?
      전수검사 / Lot당 N점 샘플 / AQL Level / Shift별 기준 검증 등
      → 없으면: 공정 생산량과 리스크 수준에 맞게 설정

□ 4. 불합격(NG) 처리 방법이 명시되어 있는가?
      자동 NG 배출 / 후공정 이동 불가 인터락 / 즉시 격리 / 전수 재검 /
      출하 불가 인터락 / 반품 처리 등
      → 없으면: 반드시 추가 (검출만 되고 유출 방지 불명확)

□ 5. D값이 검사방법과 일치하는가? (4.1항 등급표 참조)
      Poka-Yoke 자동 차단 → D=1~2
      자동측정기 + 인터락 → D=2~3
      전용 게이지·장비 전수 → D=3~4
      검교정 보증 계측기 → D=5
      한계견본 기준 전수 육안 → D=6
      일반 육안·비전(MSA 미입증) → D=7
      → 불일치 시 D값 재산정

□ 6. PC 성격의 내용이 장치명 없이 혼입되지 않았는가?
      교육 강화 / 검교정 주기(단독) / 예방보전 / 조건 최적화 /
      알림 시스템(단독) / 설계 변경(단독) / 빈도 증가(단독)
      → 장치명 없이 이런 표현만 있으면 장치명 추가 또는 PC 필드로 이동
```

## 8. 작성 금지사항

### 8.1 절대 금지 표현

| # | 금지 패턴 | 사유 | 대체 방법 |
|---|---------|------|---------|
| 1 | "검사 강화" | 장치명 없음, 추상적 | 장치명 + 전수/AQL + NG 처리 명시 |
| 2 | "Gage R&R 불합격 시 재측정" (장치명 없음) | 장치명 불명확 | "마이크로미터 Gage R&R 불합격 시 마이크로미터 재측정" |
| 3 | "마스터 샘플 검증" (장치명 없음) | 장치명 불명확 | "EOL 기능검사기 마스터 샘플 검증" |
| 4 | "검교정 이력 전산화 및 알림 도입" | PC 성격, 장치명 없음 | 계측기명 + 만료 시 대체 계측기 재측정 프로세스 |
| 5 | "점검 주기 단축" (장치명 없음) | PC 성격 | 장치명 + 마스터 샘플 검증 + NG 시 Shift 재검 |
| 6 | "설계 변경" (단독) | DC 아님 | 변경 후 적용 검사장치·성적서 Gate 관리로 기술 |
| 7 | "교육 강화" | PC 성격 | 한계견본 기준 전수 육안검사로 교체 |

### 8.2 D값 평가 금지사항

- **육안검사 → D=1~5 금지**: 사람 의존 검사의 최소 D값은 D=6
- **비전검사기 단독(MSA 미입증) → D=3~5 금지**: MSA 미입증 시 D=7
- **장치 변경 없이 D값만 저감 금지**: 반드시 장치 또는 방법 변경 근거 필요
