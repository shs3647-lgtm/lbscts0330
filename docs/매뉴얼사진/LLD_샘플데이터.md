# LLD 샘플데이터 (교훈학습 데이터베이스)

> LLD(Lessons Learned Database)는 과거 품질 이슈와 개선 사례를 축적하여 새로운 FMEA에 자동 반영하는 데이터베이스입니다.

---

## 1. LLD란?

LLD는 과거에 발생한 품질 문제(클레임, 현장 이슈, 개선 사례)를 체계적으로 기록하여,
새로운 제품/공정의 FMEA 작성 시 **자동으로 관련 교훈을 추천**해주는 시스템입니다.

### 활용 흐름
```
과거 품질 이슈 발생
    ↓
LLD에 등록 (고장형태, 원인, 개선대책, 관리값)
    ↓
새 FMEA 워크시트 작성 시
    ↓
시스템이 공정명/고장형태 기준으로 자동 매칭
    ↓
예방관리(PC) 또는 검출관리(DC) 자동 추천
```

---

## 2. LLD 데이터 필드

### 필수 필드

| 필드명 | 설명 | 예시 |
|--------|------|------|
| lldNo | LLD 관리번호 | LLD26-001 |
| classification | 분류 | RMA, ABN, CIP, ECN, FieldIssue, DevIssue |
| applyTo | 적용 대상 | prevention(예방관리) / detection(검출관리) |
| processNo | 공정번호 | OP10, OP20 |
| processName | 공정명 | 프레스, 용접, 도장 |
| productName | 제품명 | 브레이크 패드, ECU 하우징 |
| failureMode | 고장형태 | 치수불량, 용접강도 부족, 도장 벗겨짐 |
| cause | 발생원인 | 금형 마모, 용접 전류 부족, 전처리 불량 |
| improvement | 개선대책 | 금형 교체주기 단축, 전류 모니터링, 탈지 강화 |

### 관리 기준값

| 필드명 | 설명 | 예시 |
|--------|------|------|
| occurrence | 발생도 O값 (예방관리 시) | 3 (Poka-Yoke 적용) |
| detection | 검출도 D값 (검출관리 시) | 4 (SPC 모니터링) |

### 메타 정보

| 필드명 | 설명 | 예시 |
|--------|------|------|
| vehicle | 차종 | 현대 아반떼, 기아 K5 |
| target | 대상 | 설계 / 부품 / 제조 |
| m4Category | 4M 분류 | MN(사람), MC(설비), IM(부자재), EN(환경) |
| status | 상태 | R(미완료), Y(진행중), G(완료) |
| sourceType | 출처 | manual(수동등록), import(가져오기), industry(산업공통) |
| priority | 우선순위 | 숫자 (높을수록 우선 추천) |

## 3. LLD 분류 코드

| 코드 | 의미 | 설명 |
|------|------|------|
| RMA | 반품 분석 | 고객 반품 건에 대한 분석 결과 |
| ABN | 이상 발생 | 공정 중 발생한 이상/불량 |
| CIP | 지속적 개선 | 품질 개선 활동 결과 |
| ECN | 설계 변경 | 설계 변경에 따른 교훈 |
| FieldIssue | 필드 이슈 | 시장/현장에서 발생한 품질 문제 |
| DevIssue | 개발 이슈 | 개발 단계에서 발견된 문제 |

---

## 4. LLD 매칭 로직

시스템이 FMEA 워크시트의 현재 행과 LLD를 매칭하는 방법:

### 매칭 기준 (정확 매칭)
1. **공정번호** (processNo) 일치
2. **공정명** (processName) 일치
3. **제품명** (productName) 일치

### 매칭 결과
- 일치하는 LLD가 있으면 워크시트에서 자동 추천
- **예방관리용 LLD** → PC(예방관리) 입력란에 추천
- **검출관리용 LLD** → DC(검출관리) 입력란에 추천

### 수평전개
- 유사 제품의 LLD도 참고할 수 있도록 **수평전개** 기능 제공
- 제품명이 다르더라도 공정이 유사하면 교훈을 전개 가능

## 6. LLD 관리 화면

SMART FMEA 시스템에서 LLD 관리 위치:

- **등록**: 마스터데이터 > LLD 관리 > 신규 등록
- **Import**: 마스터데이터 > LLD 관리 > 엑셀 Import
- **자동 추천**: 워크시트 > ALL탭 > 5단계 리스크분석 > 개선추천 버튼
- **필터링**: PFMEA/DFMEA > LLD 페이지 > 공정별 필터
