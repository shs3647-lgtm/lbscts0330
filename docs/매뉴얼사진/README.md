## 화면 캡처 관리 규칙

1. **최신본만 유지** — 화면이 변경되면 기존 캡처를 덮어쓰기
2. **파일명 규칙** — `{번호}_{화면명}.png` (예: `04_로그인_화면.png`)
3. **저장 위치** — 모든 캡처는 `화면캡처/` 폴더에 보관
4. **매뉴얼 참조** — `02_사용자매뉴얼.md`에서 상대경로로 참조

## 매뉴얼 업데이트 방법

화면이 변경되었을 때:
1. 해당 화면 캡처 → `화면캡처/` 폴더에 저장 (기존 파일 덮어쓰기)
2. `02_사용자매뉴얼.md`에서 해당 섹션 내용 업데이트
3. 변경 내용 커밋
