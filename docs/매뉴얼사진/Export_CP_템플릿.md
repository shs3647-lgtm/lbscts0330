# Export / CP 템플릿

> SMART FMEA 시스템의 Export 및 관리계획서(CP) 출력 형식 설명

---

## 1. FMEA 워크시트 Export

### Export 형식
- **파일 형식**: Excel (.xlsx)
- **Export 위치**: 워크시트 > 메뉴바 > "Export" 버튼

### Export 내용

| 시트 | 내용 |
|------|------|
| 워크시트 | 전체 FMEA 분석 데이터 (7단계 구조) |
| 기초정보 | 프로젝트 등록 정보 |

### 워크시트 Export 컬럼 구성

```
[1단계: 구조분석]
  L1(완제품 공정) | L2(공정번호/공정명) | L3(작업요소/4M)

[2단계: 기능분석]
  L1 기능/요구사항 | L2 기능/제품특성 | L3 기능/공정특성 | 특별특성

[3단계: 고장분석]
  고장영향(FE) | 심각도(S) | 고장형태(FM) | 고장원인(FC)

[4단계: 고장연결]
  FE-FM-FC 연결 관계

[5단계: 리스크분석]
  예방관리(PC) | 발생도(O) | 검출관리(DC) | 검출도(D) | AP

[6단계: 최적화]
  권장조치 | 담당자 | 기한 | 개선 후 S'/O'/D'/AP'

[7단계: 결과문서]
  완료일 | 상태 | 비고
```

## 2. 관리계획서 (Control Plan) 템플릿

### CP 문서 구조

관리계획서는 FMEA 분석 결과를 **양산 관리**에 적용하는 문서입니다.

### CP 헤더 정보

| 항목 | 설명 |
|------|------|
| CP 번호 | 관리계획서 고유번호 (cp26-p001 등) |
| 프로젝트명 | 제품/프로젝트명 |
| 부품명/번호 | 관리 대상 부품 |
| 개정번호 | Rev.A, Rev.B... |
| 고객명 | 납품 고객 |
| 작성자/승인자 | 문서 책임자 |
| 연동 PFMEA | 원본 PFMEA ID |

### CP 항목 컬럼

| # | 컬럼명 | 설명 | FMEA 연동 |
|:-:|--------|------|----------|
| 1 | 공정번호 | OP10, OP20... | L2 구조 |
| 2 | 공정명/설명 | 공정 상세 설명 | L2 구조 |
| 3 | 설비/장치 | 사용 설비 | L3 구조 |
| 4 | 제품특성 | 관리 대상 특성 | L2 기능 |
| 5 | 공정특성 | 공정 파라미터 | L3 기능 |
| 6 | 특별특성 | CC/SC 표시 | 기능분석 |
| 7 | 규격/공차 | 관리 기준값 | - |
| 8 | 측정방법 | 검사/측정 방법 | 검출관리(DC) |
| 9 | 시료크기 | 검사 시료 수 | - |
| 10 | 검사빈도 | 검사 주기 | - |
| 11 | 관리방법 | 관리도, SPC 등 | 예방관리(PC) |
| 12 | 담당 | 책임자 | - |
| 13 | 반응계획 | 이상 시 조치 | - |

### CP-FMEA 연동

관리계획서는 FMEA와 자동 연동됩니다:

```
FMEA 워크시트
  └─ 공정(L2) + 작업요소(L3) + 특성 + SOD + AP
       ↓ 자동 연동
관리계획서(CP)
  └─ 공정 + 특성 + 규격 + 검사방법 + 관리방법 + 반응계획
```

- FMEA에서 AP=H인 항목은 CP에서 **강화된 관리** 적용
- FMEA 변경 시 CP에 **동기화 알림** 표시

## 3. PFD (공정흐름도) Export

### PFD 문서 구조

| 컬럼 | 설명 |
|------|------|
| 공정번호 | OP10, OP20... |
| 공정명 | 공정 설명 |
| 작업요소 | 세부 작업 |
| 설비/장치 | 사용 장비 |
| 제품특성 | 관리 대상 |
| 공정특성 | 공정 파라미터 |
| 특별특성 | CC/SC |

### PFD-FMEA-CP 삼각 연동

```
PFD (공정흐름도)
  ↕ 공정 구조 공유
FMEA (고장분석)
  ↕ 분석 결과 반영
CP (관리계획서)
```

세 문서는 동일한 공정 구조를 공유하며, 하나가 변경되면 나머지에 동기화됩니다.

---

## 4. 출력/다운로드 방법

| 문서 | 위치 | 버튼 |
|------|------|------|
| FMEA 워크시트 | 워크시트 > 메뉴바 | Export |
| 관리계획서 | CP > 메뉴바 | Export |
| 공정흐름도 | PFD > 메뉴바 | Export |

> 모든 Export는 `.xlsx` 형식으로 다운로드됩니다.
