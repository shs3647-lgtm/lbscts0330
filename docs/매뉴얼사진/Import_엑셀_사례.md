# Import 엑셀 사례

> SMART FMEA 시스템에서 사용하는 Import 엑셀 템플릿 형식 설명

---

## 1. PFMEA Import 엑셀

### 시트 구성

| 시트명 | 내용 | 필수 |
|--------|------|:----:|
| 기초정보 | 프로젝트 기본 정보 (회사명, 고객, 부품 등) | O |
| 워크시트 | FMEA 분석 데이터 (구조-기능-고장-리스크) | O |

### 기초정보 시트 형식

기초정보 시트는 **항목-값** 쌍으로 구성:

| 행 | A열 (항목) | B열 (값) |
|:--:|----------|---------|
| 1 | 회사명 | (주)현대모비스 |
| 2 | 설계책임 | 품질보증팀 |
| 3 | 고객 | 현대자동차 |
| 4 | 모델연도 | 2026 |
| 5 | 주제 | 브레이크 패드 |
| 6 | FMEA 시작일 | 2026-01-15 |
| 7 | FMEA 개정일 | 2026-03-08 |
| 8 | 부품명 | 브레이크 패드 어셈블리 |
| 9 | 부품번호 | BP-2026-001 |
| 10 | 기밀등급 | Master |

### 워크시트 시트 형식 (주요 컬럼)

| 컬럼 | 내용 | 설명 |
|------|------|------|
| A | 공정번호 | OP10, OP20 등 |
| B | 공정명 | 프레스, 용접, 도장 등 |
| C | 작업요소 | 소재투입, 위치결정 등 |
| D | 4M | MN(사람)/MC(설비)/IM(부자재)/EN(환경) |
| E | 기능(L2) | 공정의 기능 설명 |
| F | 제품특성 | 치수, 강도, 외관 등 |
| G | 기능(L3) | 작업요소의 기능 |
| H | 공정특성 | 온도, 압력, 시간 등 |
| I | 특별특성 | 기호 (없음/CC/SC 등) |
| J | 고장영향(FE) | 최종 고객에 미치는 영향 |
| K | 심각도(S) | 1~10 |
| L | 고장형태(FM) | 기능 미수행 방식 |
| M | 고장원인(FC) | 고장의 근본 원인 |
| N | 예방관리(PC) | 현재 예방 관리 방법 |
| O | 발생도(O) | 1~10 |
| P | 검출관리(DC) | 현재 검출 관리 방법 |
| Q | 검출도(D) | 1~10 |
| R | AP | H/M/L (자동 계산) |

## 2. DFMEA Import 엑셀

DFMEA는 PFMEA와 유사한 구조이며 다음이 다릅니다:

| 항목 | PFMEA | DFMEA |
|------|-------|-------|
| L1 | 완제품 공정 | 시스템 |
| L2 | 공정 | 서브시스템 |
| L3 | 작업요소 (4M) | 부품/컴포넌트 |
| 특성 | 공정특성 | 설계특성 |

---

## 3. Import 주의사항

### 필수 규칙
1. **엑셀 형식**: `.xlsx` (`.xls` 미지원)
2. **시트명**: 정확히 "기초정보"와 "워크시트" (대소문자 구분)
3. **빈 행**: 데이터 중간에 빈 행이 있으면 해당 행에서 읽기 중단
4. **병합 셀**: 지원됨 (상위 셀 값이 하위 셀에 자동 적용)

### 공정번호 규칙
- `OP10`, `OP20` 형식 권장
- 숫자 증분: 10단위 (OP10, OP20, OP30...)
- 세부 공정: `OP10-1`, `OP10-2` 형식 허용

### 4M 분류 코드
| 코드 | 의미 | 예시 |
|------|------|------|
| MN | Man (사람) | 작업자, 교육 |
| MC | Machine (설비) | 프레스, 로봇, 용접기 |
| IM | Material (부자재) | 접착제, 용접봉, 세정액 |
| EN | Environment (환경) | 온도, 습도, 분진 |

### 특별특성 기호
| 기호 | 의미 |
|------|------|
| (없음) | 일반 특성 |
| CC | Critical Characteristic (안전/법규) |
| SC | Significant Characteristic (기능) |
| HI | High Impact (품질 영향 높음) |

---

## 4. 엑셀 템플릿 다운로드

시스템 내에서 템플릿 다운로드:
- PFMEA > Import > "템플릿 다운로드" 버튼
- DFMEA > Import > "템플릿 다운로드" 버튼

> 실제 엑셀 템플릿 파일은 `docs/매뉴얼사진/템플릿/` 폴더에 보관됩니다.
> 사용자가 시스템에서 직접 다운로드하는 것을 권장합니다.
