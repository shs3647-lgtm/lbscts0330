# SOD 평가기준표 (AIAG-VDA FMEA 핸드북 기준)

> SMART FMEA 시스템에서 사용하는 심각도(S), 발생도(O), 검출도(D) 점수 기준

---

## 1. 심각도 (Severity, S) — 고장영향의 심각한 정도

| 점수 | 영향 범위 | 설명 | 예시 |
|:----:|----------|------|------|
| **10** | 안전/법규 (경고 없음) | 잠재적 안전 문제. 경고 없이 법규/규격 위반 | 브레이크 실패, 화재 위험 |
| **9** | 안전/법규 (경고 있음) | 잠재적 안전 문제. 경고와 함께 법규/규격 위반 | 에어백 경고등 점등 후 미작동 |
| **8** | 기능 상실 | 차량/제품 작동 불능. 고객 매우 불만족 | 엔진 시동 불가, 제품 사용 불가 |
| **7** | 기능 저하 | 차량/제품 작동 가능하나 성능 저하. 고객 불만족 | 출력 저하, 소음 증가 |
| **6** | 편의 기능 상실 | 편의 기능 작동 불가. 고객 불편 경험 | 에어컨 고장, 오디오 불량 |
| **5** | 편의 기능 저하 | 편의 기능 성능 저하. 고객 약간 불만족 | 에어컨 풍량 부족 |
| **4** | 외관/소음 | 대부분의 고객이 인지하는 외관/소음 결함 | 도장 흠집, 이음 발생 |
| **3** | 외관/소음 (경미) | 일부 고객이 인지하는 경미한 결함 | 미세 도장 불량 |
| **2** | 영향 거의 없음 | 극소수 고객만 인지. 영향 미미 | 미세 치수 편차 |
| **1** | 영향 없음 | 고장이 발생해도 영향 없음 | 고객 인지 불가 |

## 2. 발생도 (Occurrence, O) — 고장원인의 발생 가능성

| 점수 | 발생 확률 | PPM (백만개당) | 설명 | 예방관리 수준 |
|:----:|----------|:-------------:|------|-------------|
| **10** | 매우 높음 | >= 100,000 | 새로운 기술. 예방관리 없음 | 관리 방법 없음 |
| **9** | 높음 | 50,000 | 새로운 기술. 예방관리 미흡 | 검증되지 않은 관리 |
| **8** | 높음 | 20,000 | 유사 공정 경험 있으나 관리 불충분 | 기본적 관리만 존재 |
| **7** | 보통 높음 | 10,000 | 유사 공정. 가끔 발생 | 관리 있으나 효과 불확실 |
| **6** | 보통 | 2,000 | 유사 공정. 드물게 발생 | 관리 중이나 개선 필요 |
| **5** | 보통 낮음 | 500 | 동일 공정. 교육/표준 관리 | 교육 기반 관리 |
| **4** | 낮음 | 100 | 동일 공정. 검증된 관리 방법 | SPC 등 통계적 관리 |
| **3** | 매우 낮음 | 10 | 동일 공정. 다중 예방관리 | 실수방지(Poka-Yoke) 적용 |
| **2** | 극히 낮음 | 1 | 동일 공정. 완전한 예방관리 | 자동 실수방지 시스템 |
| **1** | 거의 불가능 | 0 | 발생 불가능 (설계적 제거) | 설계적으로 원인 제거 |

## 3. 검출도 (Detection, D) — 고장을 검출할 수 있는 능력

| 점수 | 검출 능력 | 설명 | 검출 방법 |
|:----:|----------|------|----------|
| **10** | 검출 불가 | 어떤 검출 방법도 없음 | 관리 방법 없음 |
| **9** | 매우 희박 | 고장 모드를 검출할 가능성이 매우 낮음 | 간접 검사만 가능 |
| **8** | 희박 | 검출 가능성 낮음 | 육안 검사 (주관적) |
| **7** | 매우 낮음 | 검출 가능성 매우 낮음 | 2중 육안 검사 |
| **6** | 낮음 | 검출 가능성 낮음 | 표준 검사 (수작업) |
| **5** | 보통 | 검출 가능성 보통 | 측정 장비 + 수동 판정 |
| **4** | 보통 높음 | 검출 가능성 높음 | SPC/관리도 모니터링 |
| **3** | 높음 | 검출 가능성 매우 높음 | 자동 비전 검사 |
| **2** | 매우 높음 | 거의 확실히 검출 | 자동 검사 + 자동 정지 |
| **1** | 확실 | 100% 검출 보장 | 설계적으로 불량 발생 불가 |

---

## 4. 점수 입력 시 참고사항

### 심각도(S)
- 고장영향(FE)에 대해 평가
- **같은 고장형태(FM)에 연결된 모든 고장원인(FC)은 동일한 S값** 적용
- S=9~10인 경우 안전/법규 관련 특별특성 지정 검토

### 발생도(O)
- 고장원인(FC)별로 개별 평가
- 예방관리(PC) 방법의 효과에 따라 결정
- **교육/표준작업만 있으면 O=5**, Poka-Yoke 적용 시 O=3

### 검출도(D)
- 검출관리(DC) 방법의 효과에 따라 결정
- **육안 검사만 있으면 D=7~8**, 자동 비전 검사 적용 시 D=3
- D=1은 설계적으로 불량 자체가 불가능한 경우에만 사용
