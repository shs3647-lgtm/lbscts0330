# STEP B 전처리 기능 가이드

> **버전**: v1.0.0 | **생성일**: 2026-03-05
> **목적**: 레거시 PFMEA STEP B 엑셀을 Smart FMEA Import 형식으로 자동 변환하여 Import 에러 제로화

---

## 1. 개요

### 1.1 전처리란?

기존 레거시 FMEA 시스템에서 작성된 STEP B 엑셀 파일을 Smart FMEA의 표준 Import 형식으로 **자동 변환**하는 기능입니다.

| 구분 | 설명 |
|------|------|
| **입력** | STEP B .xlsx 파일 (레거시 FMEA 양식) |
| **출력** | Smart FMEA DB (ImportedFlatData + MasterFailureChain) |
| **위치** | 등록 페이지 → 기초정보 템플릿 → **전처리** 탭 (4번째) |

### 1.2 탭 배치 이유

```
기존 데이터 → 수동 템플릿 → 자동 템플릿 → 전처리
(일반)        (일반)         (일반)         (특수)
```

- 기존/수동/자동: Smart FMEA에서 새로 작성하는 **일반적 흐름**
- 전처리: 레거시 STEP B 데이터가 있는 **특수 상황**에서만 사용
- 사용 빈도가 낮은 특수 기능이므로 4번째 배치

## 3. 전처리 파이프라인 상세

### 3.1 자동 변환 파이프라인

```
STEP B .xlsx 업로드
  │
  ├─ 1. 헤더 자동감지
  │     22개 컬럼 키워드 기반 위치 탐색
  │     "고장형태" + "고장원인" 동시 존재 행 감지
  │     실패 시 fallback 기본값 적용
  │
  ├─ 2. 행 파싱
  │     carry-forward: 병합셀 누락 자동 보완
  │     prefix 제거: "10번_MC_01금형" → "금형"
  │     FE 코드 제거: "Y1-6:선별재작업" → "선별재작업"
  │     scope 정규화: "Your Plant" → "YP"
  │     S값 충돌 감지: 같은 FM에 S 여러개 → 최대값
  │     D=1 육안검사 AIAG-VDA 위반 경고
  │
  ├─ 3. Import 데이터 빌드
  │     A1 공정번호, A2 공정명
  │     A3 공정기능 (없으면 자동생성)
  │     A4 제품특성, A5 고장형태
  │     B1 작업요소, B2 요소기능 (MC/MN 기능동사 분리)
  │     B3 공정특성, B4 고장원인
  │     C1 구분 (YP/SP/USER), C4 고장영향
  │     01번 공통공정 자동 삽입
  │     FC 고장사슬 (FE↔FM↔FC + PC/DC/S/O/D/AP)
  │
  ├─ 4. 교차 검증 (V1~V4)
  │     V1: C4 FE 수 = FC 고유 FE 수
  │     V2: A5 FM 수 = FC 고유 FM 수
  │     V3: B4 FC 수 = FC 고유 FC 수
  │     V4: B2 MC/MN 동일기능 중복
  │
  └─ 5. DB 저장
        saveMasterDataset() API 호출
        flatData[] + failureChains[] 저장
```

### 3.3 경고 레벨

| 레벨 | 의미 | 예시 |
|------|------|------|
| **ERROR** | 변환 실패 가능 | 헤더 감지 실패, 데이터 없음 |
| **WARN** | 이상값 발견 (변환은 진행) | prefix 오염, S값 충돌, D=1 육안검사 |
| **INFO** | 정보성 | 파싱 완료, A3 자동생성, 검증 결과 |

## 5. 제약사항

| # | 제약 | 이유 |
|---|------|------|
| 1 | **.xlsx만 지원** (.xls 불가) | ExcelJS 라이브러리 제한 — Excel에서 .xlsx로 저장 후 사용 |
| 2 | **첫 번째 시트만 파싱** | STEP B는 단일 시트 구조 |
| 3 | **FC 없는 행은 스킵** | FC(고장원인)가 데이터 행의 필수 조건 |
| 4 | **S값 충돌 시 최대값 채택** | AIAG-VDA 보수적 접근 (경고 표시) |
| 5 | **01번 공통공정 자동 삽입** | Smart FMEA 표준 구조 요구사항 |
