# AIAG-VDA FMEA AP(Action Priority) 판정 테이블

> **출처**: AIAG & VDA FMEA Handbook 1st Edition (2019)
> **용도**: S(심각도) × O(발생도) × D(검출도) 조합으로 AP(조치우선순위) 자동 산출
> **PFMEA 원칙**: S는 설계 영역 → PFMEA에서 변경 불가. O(예방관리)와 D(검출관리)만 개선 가능.

---

## 1. AP 등급 정의

| AP | 의미 | 조치 |
|----|------|------|
| **H** (High) | 높은 우선순위 | 반드시 개선 조치 필요 (O 또는 D 저감) |
| **M** (Medium) | 중간 우선순위 | 개선 조치 권장 |
| **L** (Low) | 낮은 우선순위 | 현재 수준 유지 가능 |

## 3. calculateAP() 구현 로직

```typescript
function calculateAP(s: number, o: number, d: number): 'H' | 'M' | 'L' | '' {
  if (!s || !o || !d) return '';
  if (s < 1 || s > 10 || o < 1 || o > 10 || d < 1 || d > 10) return '';

  // S = 1 → 항상 L
  if (s === 1) return 'L';

  // S = 2-3
  if (s >= 2 && s <= 3) {
    if (o >= 8) {
      if (d >= 7) return 'M';
      if (d >= 5) return 'M';
      return 'L';
    }
    return 'L';
  }

  // S = 4-6
  if (s >= 4 && s <= 6) {
    if (o >= 8) {
      if (d >= 5) return 'H';
      if (d >= 2) return 'M';
      return 'M';
    }
    if (o >= 6) {
      if (d >= 7) return 'M';
      if (d >= 2) return 'M';
      return 'L';
    }
    if (o >= 4) {
      if (d >= 7) return 'M';
      return 'L';
    }
    return 'L';
  }

  // S = 7-8
  if (s >= 7 && s <= 8) {
    if (o >= 8) return 'H';
    if (o >= 6) {
      if (d >= 2) return 'H';
      return 'M';
    }
    if (o >= 4) {
      if (d >= 7) return 'H';
      if (d >= 2) return 'M';
      return 'M';
    }
    if (o >= 2) {
      if (d >= 5) return 'M';
      return 'L';
    }
    return 'L';
  }

  // S = 9-10
  if (s >= 9) {
    if (o >= 4) {
      if (d >= 2) return 'H';
      return 'M';
    }
    if (o >= 2) {
      if (d >= 7) return 'H';
      if (d >= 5) return 'M';
      return 'L';
    }
    return 'L';
  }

  return 'L';
}
```

## 4. AP 개선 방향 (PFMEA 전용)

| 현재 AP | 개선 목표 | 방법 | LLD 구분 |
|---------|----------|------|---------|
| **H → M** | O 저감 | 예방관리(B5) 강화 | 예방관리 |
| **H → M** | D 저감 | 검출관리(A6) 강화 | 검출관리 |
| **H → M/L** | O + D 동시 저감 | PC + DC 동시 강화 | 예방관리+검출관리 |
| **M → L** | O 또는 D 저감 | PC 또는 DC 보강 | 예방관리 / 검출관리 |
| **L (S≥7)** | 유지 모니터링 | 현 수준 유지 + 관찰 | - |

> **★ PFMEA 절대 규칙**: S(심각도)는 설계 변경 없이 낮출 수 없음.
> PFMEA 영역에서는 O(발생도 → 예방관리 강화)와 D(검출도 → 검출관리 강화)만 개선 가능.

## 5. 빠른 참조 요약

```
S=9-10: O≥4이면 거의 H, O=2-3+D≥7이면 H, O=2-3+D=5-6이면 M, 나머지 L
S=7-8:  O≥8이면 H, O=6-7+D≥2이면 H, O=4-5+D≥7이면 H, O=4-5+D≥2이면 M
S=4-6:  O≥8+D≥5이면 H, O≥6+D≥2이면 M, O=4-5+D≥7이면 M, 나머지 L
S=2-3:  O≥8+D≥5이면 M, 나머지 L
S=1:    항상 L
```
