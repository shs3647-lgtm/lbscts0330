## 2. DC추천2 적용 원칙

### 2.1 3단계 체계 (모든 카테고리 공통)

```
① 모니터링(감시)  → ② 이상 발생 시 라인 정지  → ③ 제품 라인에서 격리(제거)
```

- **①**: 고장원인이 되는 공정 파라미터/상태를 자동으로 실시간 감시
- **②**: 기준 이탈 또는 이상 발생 시 공정을 자동 정지 (인터록)
- **③**: 이상 구간에서 생산된 제품을 라인에서 자동 격리/배출

### 2.2 D등급별 성숙도 태그

| D 범위 | 태그 | 의미 |
|--------|------|------|
| 8~10 | `[검출체계 구축]` | 검출 시스템 자체가 없거나 미검증 → 신규 구축 필요 |
| 6~7 | `[효과성 검증]` | 시스템 존재하나 효과성 미검증 → 유효성 확인 필요 |
| 4~5 | `[자동분류/정지]` | 검증된 검출 + 분류/정지 기능 고도화 |
| 3 | `[자동배출]` | 자동배출 수준 → 에러프루프 전환 |
| 1~2 | `[에러프루프]` | 에러프루프/원천차단 유지 + 정기 유효성 검증 |

### 2.3 사람 요소 배제

- 교육, 숙련도, 작업자 역량인증, 훈련 등 인적 관리 방법 **미적용**
- 사람 기인(MN) 고장원인도 **자동화/에러프루프로 대체**하는 개선방법 제시
- 검출 성숙도 향상 = 자동/반자동/에러프루프 장치로 원인을 차단

## 3. 고장원인 카테고리 분류 규칙

고장원인(FC) + 예방관리(PC) 텍스트에서 키워드를 매칭하여 18개 카테고리로 분류.

| # | 카테고리 | 키워드 패턴 |
|---|---------|-----------|
| 1 | PARAM_DEVIATION | 설정값, 파라미터, 조건이탈, 기준이탈, 레시피, 전류, 전압, 압력, 온도이탈, 속도, 유량, 토크값 |
| 2 | WEAR_LIFE | 마모, 수명, 열화, 노후, 피로, 교체주기, 소모품, 드레싱 |
| 3 | EQUIPMENT_FAIL | 오작동, 고장, 이상, 정지, 에러, 알람, 파손, 설비변형, 설비손상 |
| 4 | CALIBRATION | 교정, 검교정, 정도, MSA, Gage, 마스터, 유효기간 |
| 5 | SENSOR_INSPECT | 센서, 카메라, 비전, 검사기, 스캐너, 인식, 바코드, EOL, 측정기, 검사오류 |
| 6 | JIG_MOLD | 지그, 금형, 클램프, 고정, 다이, 몰드, 캐비티, 치구, 가이드 |
| 7 | FASTENING | 토크, 체결, 풀림, 나사, 볼트, 너트, 조립, 삽입, 장착, 부품누락 |
| 8 | DISPENSING | 도포, 그리스, 접착, 실란트, 디스펜서, 노즐, 코팅, 도막 |
| 9 | WELDING | 용접, 너겟, 스패터, 전극, 팁, 납땜, 솔더, 브레이징 |
| 10 | MISJUDGMENT | 오판정, 오판, 합격↔불합격, 판정오류 |
| 11 | OMISSION | 누락, 미실시, 미검사, 미점검, 미체결, 미삽입, 미장착, 빠짐, 미부착 |
| 12 | MIX_UP | 이종, 혼입, 혼동, 오투입, 품번, 품명, 사양불일치, 식별 |
| 13 | HANDLING | 운반, 이송, 낙하, 충격, 적재, 자재손상, 자재파손 |

| 15 | ENVIRONMENT | 온도, 습도, 항온항습, 정전기, ESD, 이물, 분진, 먼지, 청정, 풍속, 결로, 환경 |

> **매칭 범위**: FC + PC 텍스트 결합 후 매칭
> **Fallback**: 어떤 카테고리에도 해당하지 않으면 FALLBACK 적용

## 4. 카테고리별 DC추천2 (고장원인 개선방법)

모든 개선방법은 **① 모니터링 → ② 정지 → ③ 격리** 3단계 체계로 기술.

### 4.1 설비/파라미터 계열

| 카테고리 | DC추천2 |
|---------|--------|
| **PARAM_DEVIATION** | P_공정 파라미터 실시간 자동모니터링 → 기준이탈 시 자동정지(인터록) → 해당 제품 자동격리 |
| **WEAR_LIFE** | P_소모품 잔여수명 자동카운터 모니터링 → 한계도달 시 자동정지·교체알람 → 한계초과 생산품 자동격리 |
| **EQUIPMENT_FAIL** | P_설비상태 자동감시(이상감지 센서) → 이상발생 시 자동정지 → 이상구간 생산품 자동식별·격리 |
| **CALIBRATION** | P_측정기/검사기 자동교정확인(마스터 자동검증 사이클) → 교정이탈 시 자동정지 → 미교정구간 생산품 추적·격리 |
| **SENSOR_INSPECT** | P_검출장비 자기진단(자동 정도확인) → 이상 시 자동정지 → 이상구간 생산품 자동격리 |

### 4.2 치공구/체결/도포/접합 계열

| 카테고리 | DC추천2 |
|---------|--------|
| **JIG_MOLD** | P_치공구 장착상태 자동확인(센서) → 미장착/이탈 시 공정 자동정지(인터록) → 이탈구간 제품 격리 |
| **FASTENING** | P_체결 완료 자동확인(토크/각도 센서) → 미체결·규격이탈 시 자동정지 → NG품 자동분류·배출 |
| **DISPENSING** | P_도포량/도포상태 자동계측(중량/비전) → 규격이탈 시 자동정지 → NG품 자동분류·배출 |
| **WELDING** | P_접합 파라미터 실시간 자동감시 → 규격이탈 시 자동정지·알람 → 이탈구간 제품 자동격리 |

### 4.3 판정/누락/혼입/운반 계열

| 카테고리 | DC추천2 |
|---------|--------|
| **MISJUDGMENT** | P_자동판정 시스템(센서/비전) 도입 → 판정결과 자동검증(이중확인) → 불일치 시 라인정지·재검사 |
| **OMISSION** | P_공정완료 자동확인(센서/에러프루프) → 미완료 시 다음공정 진행 자동차단(인터록) → 미처리품 자동격리 |
| **MIX_UP** | P_자재/부품 자동식별(바코드/RFID 에러프루프) → 불일치 시 공정 자동정지 → 혼입의심품 자동격리 |
| **HANDLING** | P_이송/운반 자동화(컨베이어/가이드) → 이상하중·충격 자동감지 시 정지 → 손상의심품 자동격리 |

### 4.4 셋업/환경/포장/사람/오염 계열

| 카테고리 | DC추천2 |
|---------|--------|
| **SETUP** | P_셋업값 자동입력·자동검증(레시피 관리) → 셋업이탈 시 공정 자동정지(인터록) → 초품 자동검증 후 진행 |
| **ENVIRONMENT** | P_환경조건 실시간 자동모니터링(센서) → 기준이탈 시 공정 자동정지 → 이탈구간 생산품 격리 |
| **PACKAGING** | P_자동계수/자동라벨링(중량·비전) → 수량/식별 불일치 시 자동정지 → 불일치품 자동격리 |
| **HUMAN_SKILL** | P_자동화/에러프루프 전환으로 작업자 의존도 제거 → 공정순서 인터록 → 비정상 시 자동정지·제품격리 |
| **CONTAMINATION** | P_오염도 자동모니터링(센서/파티클카운터) → 기준초과 시 자동정지 → 오염구간 생산품 격리 |

### 4.5 Fallback (미분류)

| 카테고리 | DC추천2 |
|---------|--------|
| **FALLBACK** | P_공정상태 자동모니터링 → 이상발생 시 자동정지(인터록) → 해당 제품 라인에서 자동격리 |

## 5. 스크립트 로직

### 5.1 처리 흐름

```
FA 시트 각 행 순회 (Row 2 ~ max_row)
  │
  ├── 입력 읽기: FC(16열), PC(17열), 4M(9열), D(21열)
  │
  ├── 카테고리 분류: FC+PC 결합 텍스트 → 18개 키워드 규칙 순차 매칭
  │     └── 매칭 안 되면 → FALLBACK
  │
  ├── 개선방법 조회: 카테고리 → CATEGORY_IMPROVEMENT 딕셔너리
  │
  ├── D등급 태그 부착: D값 → [D{n}:태그명]
  │
  └── DC추천2(24열) 덮어쓰기: "[태그]P_개선방법" (기존 서식 보존)
```

### 5.2 출력 형식

```
[D등급태그]P_① 모니터링 방법 → ② 이상 시 정지 조건 → ③ 제품 격리 방법
```

**예시**:
```
[D6:효과성 검증]P_검출장비 자기진단(자동 정도확인) → 이상 시 자동정지 → 이상구간 생산품 자동격리
[D1:에러프루프]P_공정 파라미터 실시간 자동모니터링 → 기준이탈 시 자동정지(인터록) → 해당 제품 자동격리
```

### 5.3 실행 방법

```bash
python3 dc_maturity_v3.py
```

- 입력: `클로드_PFMEA_기초정보_데이터_20260223_v2_7_1.xlsx`
- 출력: `클로드_PFMEA_DC추천2_성숙도향상_v3.xlsx`
- 변경 시트: FA 통합분석 — DC추천2(X열, 24열)만 수정
- UI 변경: 없음
