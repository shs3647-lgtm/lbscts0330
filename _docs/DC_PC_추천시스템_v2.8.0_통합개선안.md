```

```

## 2. DC추천1: 현행 검출관리 확인 (추천1+추천2 통합)

### 2.1 원리

**"같은 FM에서 이미 사용 중인 실제 DC 조치 텍스트를 보여준다"**

기존 v2.7.1의 "바코드 스캐너"(장비명) 대신,
A6 Pool에 있는 실제 텍스트 **"바코드 불일치 투입 차단"** 등을 반환.

### 2.2 데이터 소스

1. **FM→DC 직접매칭**: 같은 FM을 가진 모든 FC행의 DC 조치 텍스트 수집
2. **PC→DC 연계**: 같은 PC를 가진 다른 FC의 DC 조치 텍스트 추가
3. 결과를 쉼표로 연결하여 DC추천1 셀에 기입

### 2.3 알고리즘

```python
def dc_recommend_1(fm_value, pc_value, process_no):
    """DC추천1: 실제 A6 Pool 텍스트 반환"""
    dc_set = set()
    
    # 추천1: FM→DC 직접매칭
    for row in all_rows:
        if row['fm'] == fm_value and row['dc']:
            dc_set.add(row['dc'])
    
    # 추천2: PC→DC 연계
    if pc_value:
        for row in all_rows:
            if row['pc'] == pc_value and row['dc']:
                dc_set.add(row['dc'])
    
    return ', '.join(sorted(dc_set))
```

## 3. DC추천2: 검출관리 개선안 (3가지 통합)

### 3.1 원리

**"현재 검출관리를 어떻게 개선할 수 있는지 제안한다"**

3가지 개선 관점을 하나의 셀에 통합:

| 개선 관점 | 설명 | 표시 형식 |
|-----------|------|----------|
| **a) 검출도 1등급 개선** | 현재 D값에서 1등급 낮출 DC | `[D↓] 비전검사(D≥3)` |
| **b) 성숙도 개선** | AIAG 상위 성숙도 DC | `[성숙도↑] 자동검출(D≥3←D≥6)` |
| **c) 산업표준** | 미보유 산업표준 DC | `[산업] 토크검사(체결키워드)` |

### 3.5 알고리즘

```python
def dc_recommend_2(fm_value, fc_value, current_dc, current_d, rec1_dcs):
    """DC추천2: 검출관리 개선안 (3가지 통합)"""
    improvements = []
    
    cur_mat = DC_MATURITY.get(current_dc, {'proven': 8})
    
    # a) 검출도 1등급 개선: 현재D보다 1등급 낮은 DC 찾기
    for dc, mat in DC_MATURITY.items():
        if mat['proven'] < cur_mat['proven'] and dc not in rec1_dcs:
            improvements.append(f"[D↓] {dc}(D≥{mat['proven']})")
            break
    
    # b) 성숙도 개선: 현재 성숙도보다 높은 DC
    if cur_mat['proven'] >= 6:  # 육안검사 수준이면
        for dc, mat in DC_MATURITY.items():
            if mat['proven'] <= 4 and dc not in rec1_dcs:
                improvements.append(f"[성숙도↑] {dc}(D≥{mat['proven']})")
                break
    
    # c) 산업표준: 키워드 매칭으로 미보유 DC
    industry = get_industry_dc(fm_value, fc_value)
    new_industry = industry - set(rec1_dcs) - {current_dc}
    for dc in sorted(new_industry)[:2]:
        mat = DC_MATURITY.get(dc, {'proven': 5})
        improvements.append(f"[산업] {dc}(D≥{mat['proven']})")
    
    return ', '.join(improvements[:3])  # 최대 3개
```

## 4. PC추천1: 현행 예방관리 확인 (추천1+추천2 통합)

1. **FC→PC 직접매칭**: 같은 FC를 가진 모든 행의 PC 조치 텍스트
2. **FM→PC 교차매칭**: 같은 FM의 다른 FC에서 사용하는 PC 조치 텍스트 추가
3. 결과를 쉼표로 연결하여 PC추천1 셀에 기입

```python
def pc_recommend_1(fc_value, fm_value, process_no):
```

## 5. PC추천2: 예방관리 개선안 (3가지 통합)

### 5.4 알고리즘

```python
def pc_recommend_2(fc_value, fm_value, m4, current_pc, current_o, rec1_pcs):
    """PC추천2: 예방관리 개선안 (3가지 통합)"""
    improvements = []
    
    cur_mat = PC_MATURITY.get(current_pc, {'type': 'Behavioral', 'o_proven': 7})
    
    # a) 발생도 1등급 개선: 현재O보다 1등급 낮은 PC
    for pc, mat in PC_MATURITY.items():
        if mat['o_proven'] < cur_mat['o_proven'] and pc not in rec1_pcs:
            improvements.append(f"[O↓] {pc}({mat['type'][:1]},O≥{mat['o_proven']})")
            break
    
    # b) 관리유형 향상: Behavioral → Technical
    if cur_mat['type'] == 'Behavioral':
        for pc, mat in PC_MATURITY.items():
            if mat['type'] == 'Technical' and pc not in rec1_pcs:
                improvements.append(f"[유형↑] {pc}(T,O≥{mat['o_proven']})")
                break
    
    # c) 산업표준: 키워드 매칭으로 미보유 PC
    industry = get_industry_pc(fc_value, fm_value)
    new_industry = industry - set(rec1_pcs) - {current_pc}
    for pc in sorted(new_industry)[:2]:
        improvements.append(f"[산업] {pc}")
    
    return ', '.join(improvements[:3])  # 최대 3개
```
