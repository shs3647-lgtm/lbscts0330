## 2. 필드별 연계성 매트릭스

### 2.1 공정 정보 연계

```
┌─────────────────────────────────────────────────────────────────┐
│                     공정번호 (processNo)                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CP.processNo ═══════════ PFD.processNo ═══════════ FMEA.L2.no │
│       │                         │                        │      │
│       └─────────────────────────┴────────────────────────┘      │
│                     모두 동일한 값                               │
│                                                                 │
│   예: "10" = "10" = "10"                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 공정명 연계

```
┌─────────────────────────────────────────────────────────────────┐
│                     공정명 (processName)                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  CP.processName ═══════ PFD.processName ═══════ FMEA.L2.name    │
│                                                                 │
│   예: "컷팅" = "컷팅" = "컷팅"                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 설비/작업요소 연계

```
┌─────────────────────────────────────────────────────────────────┐
│                  설비/금형/JIG → 작업요소                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CP.equipment ═══════ PFD.equipment ═══════ FMEA.L3.name       │
│                              │                                  │
│                        PFD.workElement                          │
│                                                                 │
│   예: "Cutting MC" = "Cutting MC" = "Cutting MC"                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 ⚠️ 제품특성 연계 (Critical)

```
┌─────────────────────────────────────────────────────────────────┐
│                     제품특성 (productChar)                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CP.productChar ═══════ PFD.productChar ═══════ L2Function     │
│      (Col 10)              (Col 8)          .productChar        │
│                                                                 │
│   ⚠️ 주의: 절대로 L3Function에 들어가면 안 됨!                    │
│                                                                 │
│   예: "파이프 외경" = "파이프 외경" = "파이프 외경"               │
│                                                                 │
│   ✅ 올바른 값: 파이프 외경, 파이프 두께, 파이프 길이, BURR       │
│   ❌ 잘못된 값: RPM, 지그위치, 압력, 전압 (= 공정특성이어야 함)   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.5 ⚠️ 공정특성 연계 (Critical)

```
┌─────────────────────────────────────────────────────────────────┐
│                     공정특성 (processChar)                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   CP.processChar ═══════ PFD.processChar ═══════ L3Function     │
│      (Col 11)              (Col 9)          .processChar        │
│                                                                 │
│   ⚠️ 주의: 절대로 L2Function에 들어가면 안 됨!                    │
│                                                                 │
│   예: "RPM" = "RPM" = "RPM"                                     │
│                                                                 │
│   ✅ 올바른 값: RPM, 압력, 전압, 온도, 속도, 지그위치             │
│   ❌ 잘못된 값: 파이프 외경, 두께, 길이 (= 제품특성이어야 함)     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 연계성 흐름도

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           데이터 연계 흐름                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│                    ① Excel Import 또는 수동 입력                            │
│                              │                                              │
│                              ▼                                              │
│                    ┌─────────────────┐                                      │
│                    │ UnifiedProcess  │  ← 중앙 저장소                       │
│                    │     Item        │                                      │
│                    └────────┬────────┘                                      │
│                             │                                               │
│           ┌─────────────────┼─────────────────┐                             │
│           ▼                 ▼                 ▼                             │
│    ┌────────────┐    ┌────────────┐    ┌────────────┐                       │
│    │     CP     │    │    PFD     │    │    FMEA    │                       │
│    │  108 rows  │ ══ │  108 rows  │ ══ │  L2: 3개   │                       │
│    │            │    │   (동일)   │    │  L3: N개   │                       │
│    └────────────┘    └────────────┘    └────────────┘                       │
│           │                 │                 │                             │
│           └─────────────────┴─────────────────┘                             │
│                             │                                               │
│                    ② 구조연동 / 데이터동기화                                 │
│                             │                                               │
│           ┌─────────────────┴─────────────────┐                             │
│           ▼                                   ▼                             │
│    [공정번호/공정명]                    [제품특성/공정특성]                   │
│    processNo, processName               productChar → L2Function            │
│    equipment → L3                       processChar → L3Function            │
│                                                                             │

```
