## 4. 버튼 스타일

### 4.1 공통 스타일
```css
/* 라운드 버튼 */
.btn-round {
  border-radius: 0.25rem;  /* rounded */
  padding: 2px 4px;        /* py-0.5 px-1 */
  font-weight: 500;        /* font-medium */
  transition: all;
  white-space: nowrap;
}

/* 플랫 버튼 (테두리 없음) */
.btn-flat {
  border: none;
  box-shadow: none;
}
```

### 4.2 버튼 크기
| 버튼 유형 | 패딩 (px/py) |
|----------|--------------|
| 연동/이동 버튼 | `px-1 py-0.5` (4px / 2px) |
| 확정/승인 버튼 | `px-2 py-0.5` (8px / 2px) |
| 탭 버튼 | `px-2 py-1` (8px / 4px) |

## 7. 리사이즈 기능

### 7.1 컬럼 리사이즈
- **최소 너비**: 20px
- **최대 너비**: 500px
- **리사이즈 핸들**: 우측 테두리 호버 시 `col-resize` 커서
- **저장**: 리사이즈된 너비는 상태에 저장

### 7.2 헤더 고정 (Sticky Header)
```css
.header-row {
  position: sticky;
  z-index: 20;
}
.header-row-1 { top: 100px; }
.header-row-2 { top: 130px; }
.header-row-3 { top: 184px; }
```

## 8. 메인 메뉴바 구성

```
┌─────────────────────────────────────────────────────────────────┐
```
