## 1. 테스트 환경 설정

### 1.1 시스템 환경
| 항목 | 값 |
|------|-----|
| OS | Windows 10/11 |
| Node.js | v18+ |
| Next.js | 16.1.1 (Turbopack) |

| Browser | Chrome, Edge |
