## 6. PFMEA vs DFMEA 용어 매핑

| 단계 | PFMEA (공정) | DFMEA (설계) |
|------|-------------|-------------|
| 구조분석 L1 | 완제품 공정명 | **다음상위수준** |
| 구조분석 L2 | NO+공정명 | **초점요소(A'SSY)** |
| 구조분석 L2 속성 | 4M | **타입** |
| 구조분석 L3 | 작업요소 | **다음하위수준** |
| 기능분석 L1 | 완제품기능 | **다음상위수준 기능** |
| 기능분석 L2 | 공정기능 | **초점요소 기능** |
| 기능분석 L3 | 작업요소 기능 | **다음하위수준 기능** |
| 리스크분석 | 예방관리(PC) | **현재 예방관리(PC)** |
| 리스크분석 | 검출관리(DC) | **현재 검출관리(DC)** |
| 최적화 | 예방관리개선 | **설계 예방 조치** |
| 최적화 | 검출관리개선 | **설계 검출 조치** |
