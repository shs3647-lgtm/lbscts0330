## 📋 변경 요약

CP Import 페이지를 PFMEA Import와 동일한 구조로 전면 리팩토링하였습니다.

### 주요 변경 사항

| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| **탭 구조** | 전체/그룹/개별 3개 탭 | 단일 드롭다운 (PFMEA 스타일) |
| **드롭다운** | 단순 항목명 | 계층 구조 (L2/L3/L4/L5) |
| **테이블** | 그룹별 테이블 | 엑셀 스타일 단일 테이블 |
| **우측 패널** | 없음 | CP 분석 DATA 패널 추가 |
| **레이아웃** | 단일 패널 | 2패널 (좌: 입력, 우: 분석) |
