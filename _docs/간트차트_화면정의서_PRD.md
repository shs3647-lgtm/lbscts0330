## 1. 개요

### 1.1 목적
간트차트(Gantt Chart)는 APQP(Advanced Product Quality Planning) 프로젝트의 일정을 시각적으로 관리하고 추적하기 위한 도구입니다. 현대차 WBS(Work Breakdown Structure) 스타일을 반영하여 마일스톤과 태스크를 계층적으로 표시합니다.

### 1.2 주요 기능
- 프로젝트 일정 시각화 (계획/실적)
- 마일스톤 관리 (PROTO, P1, P2, PPAP, SOP)
- 드래그 앤 드롭 일정 조정
- Excel Import/Export
- 진척률 표시

### 2.2 영역별 설명

| 영역 | 설명 |
|-----|------|
| **헤더 영역** | 컬럼 토글 버튼, 뷰 모드 선택(월/주/일), Export/Import 버튼 |
| **작업 목록 영역** | WBS ID, 작업명, 기간, 담당자 등 테이블 형식 표시 |
| **타임라인 헤더** | 연도/월 표시, 수직 그리드 라인 |
| **간트 차트 영역** | 계획/실적 바, 마일스톤 라벨, 연결선 |

## 3. 데이터 구조

### 3.1 태스크 (Task) 구조

```typescript
interface Task {
  id: string;              // WBS ID (예: "0", "0.1", "1.10")
  parentId: string | null; // 부모 태스크 ID
  level: number;           // 계층 레벨 (0: Root, 1: Phase, 2: Task)
  taskName: string;        // 작업명
  planStart: string;       // 계획 시작일 (YYYY-MM-DD)
  planFinish: string;      // 계획 종료일 (YYYY-MM-DD)
  actStart: string;        // 실적 시작일
  actFinish: string;       // 실적 종료일
  state: 'G' | 'Y' | 'R';  // 상태 (Green/Yellow/Red)
  department: string;      // 담당 팀
  owner: string;           // 담당자
  path: 'C' | '';          // 크리티컬 패스 여부
  type: 'task' | 'milestone'; // 태스크 유형
  progress: number;        // 진척률 (0-100)
}
```

### 3.2 마일스톤 목록

| 마일스톤 | 설명 | 데이터 ID |
|---------|------|----------|
| **PROTO** | 프로토타입 단계 | 0.1 |
| **P1** | Pilot 1 단계 | 0.2 |
| **P2** | Pilot 2 단계 | 0.3 |
| **PPAP** | 양산 승인 프로세스 | 0.4 |
| **SOP** | 양산 시작 | 0.5 |

## 5. 사용자 인터랙션

### 5.1 드래그 앤 드롭

| 동작 | 위치 | 결과 |
|-----|------|------|
| **시작일 조정** | 바 왼쪽 끝 드래그 | planStart 변경 |
| **종료일 조정** | 바 오른쪽 끝 드래그 | planFinish 변경 |
| **실적 종료일 조정** | 실적 바 오른쪽 끝 드래그 | actFinish 변경 |

> ⚠️ **제한사항**: 하위 태스크가 있는 그룹(Root, Phase)은 드래그 불가

### 5.2 그룹 접기/펼치기

- Phase 레벨 좌측 **▼/▶** 아이콘 클릭으로 하위 태스크 표시/숨김
- 기본값: 1~4단계 접힘, 5단계 펼침

### 5.3 컬럼 토글

| 컬럼 | 기본값 | 설명 |
|-----|-------|------|
| 기간 | 표시 | 작업 기간 (일) |
| P.St | 표시 | 계획 시작일 |
| P.Fn | 표시 | 계획 종료일 |
| A.St | 숨김 | 실적 시작일 |
| A.Fn | 숨김 | 실적 종료일 |
| Team | 표시 | 담당 팀 |
| Name | 표시 | 담당자 |

## 7. 데이터 Import/Export

### 7.1 Excel Export
- 파일명 형식: `Project_Gantt_YYYYMMDD_HHmmss.xlsx`
- 포함 데이터: 모든 태스크 정보 (ID, 작업명, 일정, 담당자 등)

### 7.2 Excel Import
- 지원 형식: `.xlsx`, `.xls`
- 필수 컬럼: WBS, Task, Plan Start, Plan Finish

### 7.3 JSON Save/Load
- 파일명 형식: `Project_Gantt_YYYYMMDD_HHmmss.json`
- 전체 태스크 데이터 저장/복원
