## 3. 알려진 기술 이슈

### 3.1 이벤트 버블링 (해결됨)
- **증상**: 체크박스 클릭 시 다중선택 안 됨
- **원인**: 행 `onClick`과 체크박스 `onChange`가 동시 발생 → 2번 토글 = 원상복귀
- **해결**: 체크박스 `<td>`에 `onClick={(e) => e.stopPropagation()}` 적용
- **상태**: UserSelectModal, BizInfoSelectModal 모두 수정 완료

### 3.2 setState 클로저 narrowing (해결됨)
- **증상**: `useAllTabModals.ts`에서 `dcText`가 `never`로 추론
- **원인**: `setState` 콜백 내부에서 할당된 변수가 외부에서 `null`로 추론됨
- **해결**: `as` 타입 단언 적용
- **상태**: 수정 완료, `tsc --noEmit` 0 errors 확인

## 4. 점검 시 유의사항

1. **모달은 공유 컴포넌트**: 모달 자체를 수정할 필요 없음. 각 페이지의 `onSelect` 콜백만 점검
2. **onSelect 시그니처**: `(user: UserInfo) => void` / `(bizInfo: BizInfoProject) => void` 변경 없음
3. **다중선택은 모달 내부 기능**: 외부에서는 여전히 단일 사용자/고객정보를 `onSelect`로 받음
4. **삭제는 모달 내부에서 직접 API 호출**: 등록 페이지에 추가 코드 불필요
