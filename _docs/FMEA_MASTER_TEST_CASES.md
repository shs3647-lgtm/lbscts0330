## 2. DFMEA 등록화면 테스트

### TC-REG-001: 신규 DFMEA 등록
| 항목 | 내용 |
|------|------|
| **테스트 ID** | TC-REG-001 |
| **테스트명** | 신규 DFMEA 프로젝트 등록 |
| **사전조건** | DFMEA 등록화면 접속 |
| **테스트 단계** | 1. `/dfmea/register` 접속<br>2. 회사명 입력 (FMEA 작성 회사)<br>3. 고객명 입력 (고객사 정보)<br>4. FMEA명 입력<br>5. 시작일자/개정일자 선택<br>6. 저장 버튼 클릭 |
| **예상 결과** | - FMEA ID 자동 생성 (dfm26-p001 형식)<br>- localStorage에 저장<br>- DB에 저장<br>- 저장 완료 메시지 표시 |
| **테스트 상태** | ✅ Pass |
| **테스트 일자** | 2026-01-21 |

### TC-REG-002: 회사명 vs 고객명 구분
| 항목 | 내용 |
|------|------|

| **예상 결과** | - 회사명(companyName): FMEA 작성 회사<br>- 고객명(customerName): 고객사 정보<br>- 두 필드가 독립적으로 저장/표시 |
