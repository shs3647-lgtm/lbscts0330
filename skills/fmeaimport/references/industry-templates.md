# 산업별 FMEA 템플릿

## 개요

이 문서는 7개 산업별 FMEA 템플릿의 구조, 필수/선택 항목, 산업별 특화 항목을 정의합니다.

**기본 구조**: AIAG-VDA PFMEA 19개 시트 (pfmea-import 참조)

---

## 1. 자동차 (Automotive) 템플릿

### 1.1 기본 구조

```
L2 시트 (공정 관점) - 7개
├── A1: 공정번호 (zero-pad)
├── A2: 공정명
├── A3: 공정기능
├── A4: 제품특성
├── A5: 고장형태
└── A6: 검출관리 (선택)

L3 시트 (요소 관점) - 5개
├── B1: 작업요소
├── B2: 요소기능
├── B3: 공정특성
├── B4: 고장원인
└── B5: 예방관리 (필수)

L1 시트 (고객 관점) - 4개 + 1개 통합
├── C1: 구분 (YP/SP/USER)
├── C2: 제품기능
├── C3: 요구사항
├── C4: 고장영향
└── L1통합: C1-C4

통합 시트 - 3개
├── L2통합: A1-A6
├── L3통합: B1-B5
└── FC: 고장사슬

분석 시트 - 1개
└── FA: 통합분석
```

### 1.2 자동차 특화 항목

#### C1 자동차 구분

```
YP: "자사 공정 내 Spec Out"
    예: "높이 Spec Out으로 인한 수율 저하"

SP: "고객 납품 품질"
    예: "높이 부족으로 인한 조립 간섭 (고객 불량)"

USER: "규제/안전/특수특성"
    예: "RoHS 미준수", "특수특성 CC 부적합"
```

#### C4 특수특성 (CC/CR) 표기

```
패턴: "[현상] + [영향] + [특수특성]"

예: "높이 부족으로 인한 조립 간섭 [CC]"
    "용접 불량으로 인한 강도 저하 [CR]"

CC = Customer Characteristic (고객 요구 특성)
CR = Characterized Requirement (특성화된 요구사항)
```

#### B5 자동차 예방관리 강화

```
자동차 규정 (IATF 16949) 요구사항:

· 설계 FMEA 결과 반영
· 공정 능력 (Cpk > 1.33) 확보
· 통계적 공정 관리 (SPC)
· 예방 조치의 효과성 검증

예: "SPC 차트 도입 및 월 1회 검토"
    "설비 능력 평가 (Cpk 목표: 1.67)"
```

### 1.3 자동차 예시 공정

```
01 금형/프레스   → 10 용접   → 20 도장   → 30 조립
40 검사          → 50 포장

각 공정별 A4/A5/B4/FC 샘플 제공 (automotive-processes.md 참조)
```

## 2. 반도체 (Semiconductor) 템플릿

### 2.1 기본 구조

자동차와 동일한 19개 시트 구조.

### 2.2 반도체 특화 항목

#### B3 공정특성 + Spec

```
패턴: "[파라미터] (Spec: [하한]~[상한])"

예: "온도 (Spec: 400°C ± 10°C)"
    "압력 (Spec: 100 ± 5 mbar)"
    "시간 (Spec: 30 ± 2 min)"
```

#### B4 고장원인 + Cpk 연계

```
패턴: "[원인] (Cpk: [목표값])"

예: "온도 편차 (Cpk 목표: > 1.33)"
    "압력 편차 (현재 Cpk: 0.95 → 개선 필요)"
```

#### C3 요구사항 (엔지니어링 기준)

```
패턴: "[특성] : [최소] ~ [최대] [단위] (공정 내 Spec)"

예: "Temperature: 395~405°C"
    "Pressure: 95~105 mbar"
    "Time: 28~32 min"
```

#### FC 공정경로 매핑

```
예: 포토 리소그래피 공정
    FE: 패턴 위치 오차
    FM: 초점 거리 부족 (Focus Distance)
    FC: 렌즈 오염, 노광 강도 편차, 기울기 각도
```

### 2.3 반도체 예시 공정

```
01 웨이퍼 준비     → 10 포토리소   → 20 에칭     → 30 증착
40 CMP             → 50 메탈 처리  → 60 패키징  → 70 테스트

각 공정별 A4/A5/B3/B4/FC 샘플 제공 (semiconductor-processes.md 참조)
```

## 3. 의료기기 (Medical Device) 템플릿

### 3.1 기본 구조

자동차와 동일한 19개 시트 + **ISO 14971 위해도 평가 시트**

```
추가 시트:
└── ISO14971: 위해도 평가 매트릭스 (심각도 × 가능성)
```

### 3.2 의료기기 특화 항목

#### ISO 14971 위해도 평가

```
위해도 = 심각도 (S) × 가능성 (P)

심각도 (1~5):
  1 = 경미 (Minor): 불편함, 기능 감소
  2 = 중간 (Moderate): 부분적 기능 상실
  3 = 중대 (Major): 완전 기능 상실
  4 = 심각 (Serious): 환자 상해 또는 사망
  5 = 치명 (Critical): 광범위 환자 상해

가능성 (1~5):
  1 = 매우낮음 (Rare): < 0.1%
  2 = 낮음 (Low): 0.1% ~ 1%
  3 = 중간 (Medium): 1% ~ 5%
  4 = 높음 (High): 5% ~ 10%
  5 = 매우높음 (Very High): > 10%

위해도 점수 행렬 (1~25):
  1~3: 낮음 (Low Risk) — 현재 상태 유지
  4~8: 중간 (Medium Risk) — 위해 통제 검토
  9~15: 높음 (High Risk) — 위해 통제 필수
  16~25: 매우높음 (Very High Risk) — 즉시 조치 필수
```

#### C4 고장영향 + 위해도

```
패턴: "[현상] + [환자영향]" + 위해도 점수

예: "센서 오작동으로 인한 잘못된 진단 (S=4, P=2, Risk=8)"
    "배터리 방전으로 인한 장비 작동 불능 (S=3, P=3, Risk=9)"
```

#### B5 위해 통제 조치

```
규제 요구사항 (ISO 14971):

· 위해도 ≥ 8: 위해 통제 조치 필수
· 위해도 < 8: 위해 통제 검토 (필요시 적용)

예: "배터리 자동 알림 시스템 설치 (위해도 9 → 3으로 감소)"
    "이중 센서 독립 검증 구조 (위해도 8 → 2)"
```

#### FDA 규제 항목

```
C1 USER 구분에 FDA 요구사항 포함:
  · 510(k) 제출 항목 표기
  · 설계 리뷰 결과 반영
  · 임상 평가 데이터
```

### 3.3 의료기기 예시 공정

```
01 부품 수급   → 10 조립   → 20 세척   → 30 포장
40 멸균         → 50 품질검사 → 60 출하

각 공정별 ISO 14971 위해도 평가 (medical-processes.md 참조)
```

#### B1 전자 조립 요소

```
4M 분류:
  · Man: 작업자 (Operator ID)
  · Machine: SMT 장비, 납땜 장비, 검사 장비
  · Material: PCB, 부품(IC, Capacitor), 납땜 페이스트
  · Method: 리플로우 온도 프로파일, 납땜 시간
```

#### B3 전자 공정특성

```
예: "리플로우 최고온도 (Peak Temp)"
    "ROI 시간 (Time over 217°C)"
    "언더필 두께"
    "납땜 접합 높이"
```

#### B4 전자 고장원인

```
예: "냉접점 (Cold Joint)"
    "브릿지 (Solder Bridge)"
    "부품 탈락"
    "부품 오방향 장착"
```

#### C3 전자 요구사항

```
예: "Peak Temp: 250°C max"
    "Time above 217°C: 40~100 sec"
    "Solder Joint: 완전 습윤"
```

### 4.3 전자제품 예시 공정


## 10. 템플릿 로드 체크리스트

선택한 템플릿으로 작업 시작 전:

```
□ 산업 템플릿 SKILL.md §2 확인
□ 산업별 공정 카탈로그 (e.g., automotive-processes.md) 준비
□ 산업 특화 항목 (C1/B5/검증 규칙) 이해
□ 엑셀 템플릿 구조 (19개 시트) 준비
□ 데이터 수집 (공정/A4/B4/FC 등)
```
