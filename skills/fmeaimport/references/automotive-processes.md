# 자동차 (Automotive) 공정 카탈로그

## 개요

IATF 16949 기준 자동차 제조 공정별 FMEA 데이터 샘플.

---

## 공정 맵

```
01 설계/금형   → 10 프레스/스탬핑   → 20 용접
30 도장         → 40 조립           → 50 최종검사 → 60 포장/출하
```

---

## 01. 금형 설계 공정

### A4 제품특성 (금형 결과값)
- 금형 정밀도 (±0.05mm)
- 금형 수명 (10만회 이상)
- 금형 표면 거칠기

### A5 고장형태
- 금형 정밀도 부족
- 금형 조기 마모
- 금형 표면 손상

### B1 작업요소
- CAD 설계팀, CNC 가공기, 금형 강재, 정밀 측정

### B4 고장원인
- 설계 검토 미흡
- CNC 가공 정밀도 편차
- 강재 내재 결함
- 냉각/경화 온도 편차

### B5 예방관리
- 설계 리뷰 2회 (초안, 최종)
- CNC 검사 100% (측정기 검증)
- 금형 강재 시험성적서 확인
- 열처리 공정능력 (Cpk > 1.33)

### FC 고장사슬 예시
```
FM: 금형 정밀도 부족
 ├─ FC: CAD 설계 오류
 ├─ FC: CNC 공차 편차
 └─ FC: 경화 온도 상한 이탈
```

## 10. 프레스/스탬핑 공정

### A4 제품특성
- 부품 높이 (mm)
- 부품 두께 (mm)
- 표면 거칠기 (μm)
- 휨도 (mm)

### A5 고장형태
- 높이 부족/초과
- 두께 편차
- 표면 흠집/긁힘
- 휨/변형

### B1 작업요소
- 프레스 유압 장치, 금형 #1~3, 윤활유, 작업자

### B4 고장원인
- 유압 편차 (±5bar 이상)
- 금형 마모
- 윤활유 오염
- 금형 정렬 불량
- 작업자 조작 실수

### B5 예방관리
- 일일 유압 검사 (목표: ±3bar 이내)
- 금형 수명 관리 (모니터링 카드)
- 윤활유 교체 (매 100시간)
- 금형 정렬 자동화 센서 설치

### C1 구분 & C4 고장영향
```
YP: "높이 부족으로 인한 수율 저하"
SP: "표면 흠집으로 인한 고객 반품 [CC]"
USER: "휨 초과로 인한 안전 우려"
```

### FC 고장사슬 예시
```
FE-1: 수율 저하  )
FE-2: 고객 반품  )-- FM: 높이 부족 --┬─ FC: 유압 편차
FE-3: 재작업비   )                   ├─ FC: 금형 마모
                                     └─ FC: 윤활유 오염
```

## 20. 용접 공정

### A4 제품특성
- 용접 너비 (mm)
- 용접 깊이 (mm)
- 용접부 강도 (MPa)

### A5 고장형태
- 용접 너비 부족
- 용접 깊이 부족
- 용접부 균열
- 용접부 기공

### B1 작업요소
- 용접기 #1, 용접 전극, 냉각수, 작업자

### B4 고장원인
- 용접 전류 편차
- 용접 시간 부족
- 냉각수 온도 상한 이탈
- 전극 손상/마모
- 작업자 기술 부족

### B5 예방관리
- 용접기 월 1회 검사 (전류 측정)
- 전극 교체 관리 (생산량 기반)
- 냉각수 온도 자동 관리 (±2°C)
- 작업자 자격 관리 (연 1회 재교육)

### C4 고장영향
```
"용접 강도 부족으로 인한 차체 안전성 저하 [CR]"
```

### FC 고장사슬
```
FM: 용접 깊이 부족
 ├─ FC: 용접 전류 편차
 ├─ FC: 용접 시간 부족
 └─ FC: 전극 마모
```

## 30. 도장 공정

### A4 제품특성
- 도료 두께 (μm)
- 도료 경도 (H)
- 광택도 (%)
- 접착성

### A5 고장형태
- 도료 두께 부족
- 도료 경도 저하
- 도료 벗겨짐
- 색상 편차

### B1 작업요소
- 도장 로봇, 도료 (색상별), 건조로, 작업자

### B4 고장원인
- 스프레이 각도 편차
- 도료 점도 편차
- 건조 온도 부족
- 도료 유통기한 경과
- 표면 전처리 불량

### B5 예방관리
- 스프레이 각도 자동 조정 (정기 검사)
- 도료 점도 자동 관리 시스템
- 건조로 온도 자동 관리 (±2°C)
- 도료 입고 시 검수 및 유통기한 관리

### C4 고장영향
```
"도료 벗겨짐으로 인한 부식 및 고객 반품"
```

## 40. 조립 공정

### A4 제품특성
- 부품 결합 강도 (N)
- 조립 치수 (mm)
- 조립 시간 (min)

### A5 고장형태
- 부품 탈락
- 치수 불일치
- 조립 불량

### B1 작업요소
- 조립 지그, 부품 A/B/C, 나사/볼트, 작업자

### B4 고장원인
- 나사 조임력 부족
- 부품 위치 오류
- 부품 누락
- 부품 불량 수급

### B5 예방관리
- 토크 렌치 자동 검사 (월 1회)
- 부품 위치 자동 인식 (비전 검사)
- 부품 개수 확인 센서
- 부품 입고 검수 100%

## 50. 최종 검사 공정

### A4 제품특성
- 불량품 합격률 (%)
- 검사 시간 (min/개)

### A5 고장형태
- 검사 정확도 부족
- 검사 누락

### B1 작업요소
- 검사원, 측정 장비, 검사 기준서

### B4 고장원인
- 검사원 기술 부족
- 측정기 정확도 저하
- 기준서 미숙지

### B5 예방관리
- 검사원 월 1회 재교육
- 측정기 월 1회 검정 (외부 교정소)
- 기준서 분기별 재검토

## 특수특성 (CC/CR) 관리

### CC (Customer Characteristic)
- 도료 두께
- 용접 깊이
- 치수 정확도

### CR (Characterized Requirement)
- 강도 (용접부, 결합부)
- 내구성 (금형 수명)
- 안전성 (차체 강성)

### IATF 요구사항
```
□ CC/CR 항목은 C4에 표기 [CC] 또는 [CR]
□ CC/CR 항목은 B5 예방관리 강화
□ CC/CR 항목 능력 검증 데이터 보유 (Cpk, PPk)
□ 고객 승인 변경사항 문서화
```

## 자동차 산업 추가 체크리스트

```
□ IATF 특수특성 (CC/CR) 표기
□ 설계 FMEA 결과 반영
□ 공정능력 데이터 (Cpk > 1.33)
□ SPC 차트 (주 1회 검토)
□ 회귀방지 대책 (재발 방지)
□ 고객 승인 서명 (변경사항)
```
