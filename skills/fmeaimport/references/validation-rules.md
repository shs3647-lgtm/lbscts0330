# 검증 규칙 (25개)

## 개요

FMEA 생성 후 자동 검증 및 교정하는 25개 규칙.

---

## 구조 검증 (8개)

### 1. PROC_ZERO_PAD
**규칙**: 공정번호는 zero-padding 필수
```
오류: A1 = '1', '2', '10'
교정: '01', '02', '10'
코드: str(v).zfill(2)
```

### 2. A6_PROC_DUP
**규칙**: A6 동일 공정번호 2행 이상 금지
```
오류: 공정번호 '01'에 A6이 2개 행
교정: 두 번째 행 삭제 (또는 합치기)
```

### 3. B5_WE_EMPTY
**규칙**: B5 (예방관리) 작업요소 열 빈 셀 금지
```
오류: B5 행에 작업요소(WE) 칼럼 비어있음
교정: B1 (작업요소) 매핑으로 자동 채우기
```

### 4. L3_B5_EMPTY
**규칙**: L3 통합 시트의 B5 열 빈 셀 금지
```
오류: L3통합 B5 행 빈 셀
교정: B5 시트에서 동일 (공정번호, B4) 조합 찾아 채우기
폴백: (공정번호, B4) 키 매핑으로 B1 추론
```

### 5. L1_C3_EMPTY
**규칙**: L1 통합에서 C4 있는 행은 C2, C3 필수
```
오류: C4 기입했는데 C2/C3 빈 행
교정: C4 내용에서 C3 역매핑 (템플릿 사용)
```

### 6. C3_GROUP_ORDER
**규칙**: C3 시트 행 순서는 YP → SP → USER
```
오류: USER 그룹이 SP 앞에 위치
교정: SP 행들을 USER 앞으로 이동 (정렬)
```

### 7. FC_BALANCE
**규칙**: FM 그룹 내 FE:FC 수 균형
```
오류: FE 2개, FC 3개 (또는 역)
교정: 부족한 쪽 마지막 값 반복 행 추가
```

### 8. FC_MERGE_CHECK
**규칙**: FC 셀병합 대상 (A,B,C,D) 확인
```
오류: A열(FE 구분) 병합 안 됨
교정: 동일 FE는 셀병합 적용
```

## 내용 품질 검증 (12개)

### 9. A4_NOUN_ONLY
**규칙**: A4 (제품특성)는 명사형만
```
오류: "높이를 측정하다", "높이 > 10mm"
교정: "높이" (명사형)
```

### 10. A5_PATTERN
**규칙**: A5 = [A4] + [이탈유형]
```
오류: "고장", "불량 유출" (← C4 관점)
교정: "[A4 명사] + [부족/초과/편차]"
```

### 11. B1_UNIQUE_NAME
**규칙**: B1은 고유명, 관리항목/상태명 제외
```
오류: "온도 조절", "압력 관리" (상태명)
교정: "온도 센서 #1", "프레스 유압 장치"
```

### 12. B3_NOUN_PARAM
**규칙**: B3 (공정특성)은 파라미터 명사형
```
오류: "온도를 올린다", "온도가 높다"
교정: "온도" (설정값, 입력 파라미터)
```

### 13. B4_NOUN_PATTERN
**규칙**: B4는 명사형 3대 패턴
```
패턴 1 이탈: "[파라미터] 편차/규격 이탈"
패턴 2 인과: "[원인]에 의한 [결과]"
패턴 3 누락: "[대상] 미실시/누락"
오류: "온도가 높다" (형용사), "온도 설정 실패" (동사)
```

### 14. B4_VERB_ENDING
**규칙**: B4에서 동사형 종결 금지
```
오류: "~한다", "~않음", "~진행함"
교정: 명사형 3대 패턴으로 변환
```

### 15. C2_FUNCTION
**규칙**: C2 (제품기능)는 함수형 명사
```
패턴: "[대상] + [행위] + [결과]"
오류: "높이", "높이 기준"
교정: "웨이퍼가 규격 내 높이를 제공한다"
```

### 16. C3_MEASURABLE
**규칙**: C3 (요구사항)은 영문 단위 포함
```
오류: "높이 적절", "규격 만족"
교정: "Height: 10.0 ± 0.5 mm"
```

### 17. C4_IMPACT
**규칙**: C4 = [현상] + [고객영향]
```
오류: "불량", "고장" (단편적)
교정: "높이 부족으로 인한 조립 간섭"
        "표면 거칠기 증가로 고객 반품"
```

### 18. B5_A6_SEPARATION
**규칙**: B5 (예방)과 A6 (검출) 명확 분리
```
B5 예방: "온도 자동 조절 장치", "설비 일일 점검"
A6 검출: "온도 측정기", "매 1시간 검사"
오류: B5에 "검사 강화" (← A6)
```

## SSOT 3축 교차검증 (3개)

### 19. FC_FM_MATCH
**규칙**: set(A5.FM) == set(FC.D열.FM)
```
오류: A5에 "높이 부족" FM 있는데 FC에 없음
교정: FC 시트에 누락된 FM 추가
```

### 20. FC_B4_MATCH
**규칙**: set(FC.G열 FC) ⊆ set(B4.고장원인)
```
오류: FC에 "온도 편차" 있는데 B4에 없음
교정: B4에 "온도 편차" 추가 또는 FC 수정
```

### 21. FC_B1_MATCH
**규칙**: set(FC.F열 WE) ⊆ set(B1.작업요소)
```
오류: FC에 "노즐 #1" 있는데 B1에 없음
교정: B1에 "노즐 #1" 추가
```

## 산업별 추가 검증 (2개)

### 22. AUTOMOTIVE_CC_CR
**규칙**: 자동차 특수특성 (CC/CR) 표기 필수
```
산업: Automotive
오류: C4에 "고객 특성"인데 CC 미표기
교정: C4 끝에 [CC] 또는 [CR] 추가
```

### 23. MEDICAL_HAZARD_RISK
**규칙**: 의료기기 위해도 (Risk > 8) 검증
```
산업: Medical Device
오류: Risk = 3×3=9 인데 B5 조치 없음
교정: B5에 위해 통제 조치 추가
```

---

## 데이터 무결성 (2개)

### 24. DUPLICATE_ROWS
**규칙**: 완전 동일한 행 제거
```
오류: A1~A5 동일한 행 2개 존재
교정: 중복 행 1개 삭제
```

### 25. CELL_FORMAT_CHECK
**규칙**: 셀 포맷 통일 (텍스트 vs 숫자)
```
오류: A1 = "'01'" (텍스트), B1 = "1" (숫자)
교정: 모두 텍스트로 통일
```

---

## 검증 순서

```
Step 1. 구조 검증 (1~8번)
        └─ zero-padding, 빈 셀, 병합, 균형

Step 2. 내용 품질 (9~18번)
        └─ 명사형, 패턴, 분리

Step 3. SSOT 3축 (19~21번)
        └─ FM/B4/B1 일치성

Step 4. 산업별 (22~23번)
        └─ CC/CR, 위해도

Step 5. 최종 무결성 (24~25번)
        └─ 중복, 포맷
```

---

## 자동 교정 시 색상 표시

**적색 폰트 (FF0000)** 적용:
- 자동 추론값
- 자동 생성된 A3, A6
- SSOT 불일치로 강제 수정된 값

---

## 오류 레벨

```
CRITICAL: 교정 불가능, Import 실패
  └─ FC_FM_MATCH, FC_B4_MATCH 위반

HIGH: 자동 교정 후 사용자 검토 필수
  └─ B4_VERB_ENDING, C3_MEASURABLE 위반

MEDIUM: 자동 교정만으로 통과
  └─ PROC_ZERO_PAD, B5_WE_EMPTY

LOW: 경고만 표시
  └─ DUPLICATE_ROWS (비즈니스 의도 확인)
```
